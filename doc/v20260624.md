## 3. 积分系统（已实现）

### 3.1 核心流程

```
添加积分: SELECT member FOR UPDATE → INSERT xhbs_points_order(orgin/after)
         → INSERT xhbs_points_user_into_bill_detail → UPDATE member.current_points
         → COMMIT (悲观锁+事务保证原子性)

并发安全: 同一会员并发加积分 → 后者等待前者提交后才能读取最新积分
```

## 6. 借书业务表关系（已设计）

```
book_info ──< book_image
book_info ──< book_info_history (变更历史, source_order_no追溯)

member ──< book_borrow_order (DY) ──< book_borrow_detail
                                           ├── book_borrow_detail_image (照片)
                                           ├── book_return_detail (HS, 还书)
                                           └── dd_book_purchase_order (JSDD, 借转购)

book_borrow_order ──< book_borrow_log (trace_id)
book_return_detail ──< book_return_log (trace_id)
dd_book_purchase_order ──< book_purchase_log (trace_id)
```
