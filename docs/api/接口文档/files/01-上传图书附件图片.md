# 上传图书附件图片

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `POST /api/mp/v1/files/book-attachment-images` |
| 认证 | Bearer Token |
| Content-Type | `multipart/form-data` |
| 对应代码 | `FileController.uploadBookImage()` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| file | form-data | file | 是 | 图片文件，最大 2MB |
| memberId | form-data | string | 是 | 会员 ID |
| borrowOrderNo | form-data | string | 否 | 借阅单号 |
| borrowDetailId | form-data | long | 否 | 借阅明细 ID |
| borrowOrderId | form-data | long | 否 | 借阅单 ID |
| imageType | form-data | int | 否 | 图片类型，默认 `1` |
| bookId | form-data | long | 否 | 图书 ID |

## 完整请求报文

```http
POST /api/mp/v1/files/book-attachment-images HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <accessToken>
Content-Type: multipart/form-data; boundary=----WebKitFormBoundaryDemo

------WebKitFormBoundaryDemo
Content-Disposition: form-data; name="memberId"

1001
------WebKitFormBoundaryDemo
Content-Disposition: form-data; name="borrowOrderNo"

BR202607050001
------WebKitFormBoundaryDemo
Content-Disposition: form-data; name="borrowDetailId"

3001
------WebKitFormBoundaryDemo
Content-Disposition: form-data; name="imageType"

1
------WebKitFormBoundaryDemo
Content-Disposition: form-data; name="file"; filename="book.jpg"
Content-Type: image/jpeg

<binary>
------WebKitFormBoundaryDemo--
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data.imageId | string | 图片 ID |
| data.url | string | 原图 URL |
| data.thumbUrl | string | 缩略图 URL |
| data.size | long | 文件大小 |
| data.fileName | string | 原文件名 |

## 完整响应报文

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "imageId": "img_20260705_001",
    "url": "https://cos.example.com/bookstore/1001/book.jpg",
    "thumbUrl": "https://cos.example.com/bookstore/1001/book_thumb.jpg",
    "size": 128000,
    "fileName": "book.jpg"
  },
  "requestId": "req_20260705_110070_001"
}
```

## 异常响应

```json
{
  "code": 400,
  "message": "文件大小超过限制",
  "data": null,
  "requestId": "req_20260705_110070_002"

```
