# 上传图书附件图片

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `POST /api/mp/v1/files/book-attachment-images` |
| 认证 | Bearer Token |
| Content-Type | `multipart/form-data` |
| 对应代码 | `FileController.uploadBookImage()` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| file | form-data | file | 是 | 图片文件，最大 2MB；只允许 JPEG、PNG |
| memberId | form-data | string | 条件必填 | 会员资源上传时必填；借阅明细上传时由服务端从明细所属订单推导 |
| borrowOrderNo | form-data | string | 否 | 借阅单号 |
| borrowDetailId | form-data | long | 否 | 借阅明细 ID |
| borrowOrderId | form-data | long | 否 | 借阅单 ID |
| imageType | form-data | int | 否 | 图片类型，默认 `1` |
| bookId | form-data | long | 否 | 图书 ID |

参数组合只能选择一种资源：

- 借阅明细图片：传 `borrowDetailId`，其他订单和会员字段仅用于一致性校验。
- 图书封面：传 `bookId`，仅员工允许操作。
- 会员图片：不传明细和图书时传 `memberId`。

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data.imageId | string | 图片 ID |
| data.url | string | 原图 URL |
| data.thumbUrl | string | 缩略图 URL |
| data.size | long | 文件大小 |
| data.fileName | string | 原文件名 |
| data.format | string | 服务端识别的格式：`jpg` 或 `png` |
| data.width | int | 图片宽度，像素 |
| data.height | int | 图片高度，像素 |

## 权限和文件校验规则

- 接口必须携带有效 Bearer Token，调用者必须具备员工或会员身份。
- 会员只能上传本人资源，不能通过伪造 `memberId` 操作其他会员数据。
- 借阅图片以 `borrowDetailId` 查询真实借阅明细和订单，并校验 `memberId`、`borrowOrderId`、`borrowOrderNo` 一致性。
- 图书封面只允许员工上传；服务端校验图书真实存在。
- 不信任客户端 `Content-Type`，服务端同时校验扩展名、JPEG/PNG 文件签名和图片解码结果。
- 单文件最大 2MB，图片宽高和总像素受服务端限制；同一资源、同一图片类型最多 10 张。
- COS 上传成功但数据库记录失败时，服务端删除已上传对象，避免产生无业务归属文件。
- 成功和失败上传均记录调用身份、资源 ID、对象 Key、文件大小或失败原因；日志不记录访问令牌和密钥。

## 异常响应

```json
{
  "code": 400,
  "message": "文件大小超过限制",
  "data": null,
  "requestId": "req_20260705_110070_002"
}
```
