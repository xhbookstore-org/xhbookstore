# 门店列表

## 基本信息

| 项目 | 内容 |
|---|---|
| 接口 | `GET /api/mp/v1/stores` |
| 认证 | 不需要 |
| 对应代码 | `StoreController.stores()` |

## 业务说明

用于用户注册登录前选择绑定门店。接口只返回启用状态的部门/门店数据，前端展示 `deptName`，提交登录注册时传 `deptId`。

## 入参

无请求参数。

## 完整请求报文

```http
GET /api/mp/v1/stores HTTP/1.1
Host: 152.136.127.168
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| code | int | `0=成功` |
| message | string | 提示信息 |
| data.list | array | 门店列表 |
| data.list[].deptId | long | 门店/部门 ID |
| data.list[].deptName | string | 门店/部门名称 |
| requestId | string | 请求追踪 ID |

## 完整响应报文

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "list": [
      {
        "deptId": 101,
        "deptName": "新华书店测试门店"
      }
    ]
  },
  "requestId": "req_20260707_090030_001"
}
```
