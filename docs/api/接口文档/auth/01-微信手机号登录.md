# 微信手机号登录

> **接口路径**: `POST /api/mp/v1/auth/wechat-phone-login`
> **认证要求**: 否（白名单接口，受 IP 限流限制，默认 5 次/分钟）
> **Content-Type**: application/json;charset=UTF-8
> **对应代码**: `AuthController.java`

## 接口说明

通过微信授权 code 换取手机号，自动登录。**同时查询 `member` 表和 `sys_user` 表**，识别用户身份：
- 在 `member` 表中 → 会员（`isMember: true`）
- 在 `sys_user` 表中 → 员工（`isStaff: true`, `staffUserId` 有值）
- 两张表都有 → 既是会员又是员工
- 两张表都无 → 新访客

---

## 请求参数

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| code | String | 是 | 无 | 微信小程序端 `getPhoneNumber` 返回的动态凭证 |

## 请求报文

```json
{
  "code": "063mQn000w8h0F1QHC000EKb3o3mQn0C"
}
```

## curl 测试

```bash
curl -X POST http://localhost:8091/api/mp/v1/auth/wechat-phone-login \
  -H "Content-Type: application/json;charset=UTF-8" \
  -d '{"code":"test123"}'
```

## 响应参数

| 字段 | 类型 | 说明 |
|------|------|------|
| code | Integer | 0=成功 |
| message | String | 提示信息 |
| data.accessToken | String | 访问令牌（JWT），有效期 7200 秒（2小时） |
| data.refreshToken | String | 刷新令牌（JWT），有效期 2592000 秒（30天） |
| data.expiresIn | Long | accessToken 有效期（秒） |
| **data.isStaff** | **Boolean** | **是否员工**（在 sys_user 表中且 status=正常） |
| **data.staffUserId** | **Long** | **员工 user_id**（isStaff=true 时有值，否则 null） |
| **data.isMember** | **Boolean** | **是否会员**（在 member 表中） |
| data.userId | String | 用户唯一标识：`M{memberId}` / `S{staffUserId}` / UUID |
| data.memberId | Integer | 会员 ID（isMember=true 时有值，否则 null） |
| requestId | String | 本次请求唯一标识 |

## 业务逻辑

1. 接收微信 `code`，调用微信服务获取手机号（开发环境使用 Mock）
2. **分别查询两张表**（SQL 层已做状态过滤）：
   - `memberMapper.selectMemberByPhone(phone)` → 仅返回 `status=0`（正常）的会员
   - `sysUserMapper.selectUserByPhonenumber(phone)` → 仅返回 `status='0'` 且 `del_flag='0'` 的有效员工
3. **代码层二次校验**：会员 status 不等于 0 则视为非会员（防御性编程）
4. 汇总身份：
   - `isMember` = member 不为 null（且 status=0）
   - `isStaff` = sys_user 不为 null

> **会员状态枚举**: 0=正常(可登录), 1=注销(不可登录), 2=挂失(不可登录)
4. 生成唯一 `userId`：会员 → `M{memberId}`，员工 → `S{staffUserId}`，访客 → UUID
5. 生成双 Token（HS256 签名），JWT 中包含完整的身份信息：
   - `isMember`, `isStaff`, `phone`, `memberId`, `staffUserId`
6. 登录接口受 IP 级别限流保护
