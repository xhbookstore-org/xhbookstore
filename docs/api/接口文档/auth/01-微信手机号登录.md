# 微信手机号登录

## 基本信息

| 项目 | 内容 |
|---|---|
| 接口 | `POST /api/mp/v1/auth/wechat-phone-login` |
| 认证 | 不需要 |
| Content-Type | `application/json;charset=UTF-8` |
| 对应代码 | `AuthController.wechatPhoneLogin()` |

## 业务说明

小程序通过微信手机号授权 `code` 登录。登录注册时必须携带门店 ID，用于新用户注册绑定门店、生成门店规则会员编号。

规则：

- `deptId` 必填，且必须是启用状态门店/部门。
- 如果手机号已存在会员，且会员未绑定门店，则绑定本次传入的 `deptId`。
- 如果手机号已存在会员，且已绑定门店，不覆盖原门店。
- 如果手机号不存在会员，则自动创建普通会员，绑定 `deptId`。
- 新会员编号按门店生成：`门店编码 + 递增编号`，总长 11 位，并与已有会员编号排重。
- 如果手机号同时是员工手机号，返回 `isStaff=true` 和 `staffUserId`。

## 入参

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| code | string | 是 | 微信手机号授权 code |
| deptId | long | 是 | 用户选择绑定的门店 ID |

## 完整请求报文

```http
POST /api/mp/v1/auth/wechat-phone-login HTTP/1.1
Host: 152.136.127.168
Content-Type: application/json;charset=UTF-8

{
  "code": "wx_phone_code_demo",
  "deptId": 101
}
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| code | int | `0=成功` |
| message | string | 提示信息 |
| data.accessToken | string | 访问令牌 |
| data.refreshToken | string | 刷新令牌 |
| data.expiresIn | long | accessToken 有效秒数 |
| data.isStaff | boolean | 是否员工 |
| data.staffUserId | long | 员工用户 ID，非员工为 `null` |
| data.isMember | boolean | 是否会员 |
| data.userId | string | 统一用户标识，会员为 `M{id}`，员工为 `S{id}` |
| data.memberId | int | 会员 ID |
| data.deptId | long | 会员绑定门店 ID |
| data.needBindStore | boolean | 是否需要绑定门店；正常登录注册成功后应为 `false` |
| requestId | string | 请求追踪 ID |
