# 微信手机号登录

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `POST /api/mp/v1/auth/wechat-phone-login` |
| 认证 | 不需要 |
| Content-Type | `application/json;charset=UTF-8` |
| 对应代码 | `AuthController.wechatPhoneLogin()` |

## 入参

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| code | string | 是 | 微信手机号授权 code |

## 完整请求报文

```http
POST /api/mp/v1/auth/wechat-phone-login HTTP/1.1
Host: 152.136.127.168
Content-Type: application/json;charset=UTF-8

{
  "code": "wx_phone_code_demo"
}
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| code | int | `0=成功` |
| message | string | 提示信息 |
| data.accessToken | string | 访问令牌 |
| data.refreshToken | string | 刷新令牌 |
| data.expiresIn | long | accessToken 有效秒数 |
| data.isStaff | boolean | 是否员工 |
| data.staffUserId | long | 员工用户 ID |
| data.isMember | boolean | 是否会员 |
| data.userId | string | 统一用户标识，会员为 `M{id}`，员工为 `S{id}` |
| data.memberId | int | 会员 ID |
| requestId | string | 请求追踪 ID |

## 完整响应报文

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "accessToken": "eyJhbGciOiJIUzI1NiJ9.access",
    "refreshToken": "eyJhbGciOiJIUzI1NiJ9.refresh",
    "expiresIn": 7200,
    "isStaff": false,
    "staffUserId": null,
    "isMember": true,
    "userId": "M1001",
    "memberId": 1001
  },
  "requestId": "req_20260705_110000_001"
}
```

## 异常响应

```json
{
  "code": 400,
  "message": "缺少微信授权码",
  "data": null,
  "requestId": "req_20260705_110000_002"
}
```
