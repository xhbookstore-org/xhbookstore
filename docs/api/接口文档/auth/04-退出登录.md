# 退出登录

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `POST /api/mp/v1/auth/logout` |
| 认证 | Bearer Token |
| Content-Type | `application/json;charset=UTF-8` |
| 对应代码 | `AuthController.logout()` |

## 入参

请求体为空对象。

## 完整请求报文

```http
POST /api/mp/v1/auth/logout HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <accessToken>
Content-Type: application/json;charset=UTF-8

{}
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data.success | boolean | 是否退出成功 |

## 完整响应报文

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "success": true
  },
  "requestId": "req_20260705_110030_001"
}
```
