# 校验登录态

## 基本信息

| 项目 | 内容 |
|---|---|
| 接口 | `GET /api/mp/v1/auth/session` |
| 认证 | 可带 accessToken；不带 token 也返回 `valid=false` |
| 对应代码 | `AuthController.checkSession()` |

## Header

| 字段 | 必填 | 说明 |
|---|---|---|
| Authorization | 否 | `Bearer <accessToken>` |
| accessToken | 否 | 兼容 Header，值为登录接口返回的 `accessToken` |

## 入参

无请求参数。

## 完整请求报文

```http
GET /api/mp/v1/auth/session HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <accessToken>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data.valid | boolean | 登录态是否有效 |
| data.isStaff | boolean | 是否员工，有效登录时返回 |
| data.isMember | boolean | 是否会员，有效登录时返回 |
| data.staffUserId | long | 员工 ID |
| data.memberId | int | 会员 ID |
| data.deptId | long | 会员绑定门店 ID |
| data.userId | string | 统一用户标识 |

## 完整响应报文

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "valid": true,
    "isStaff": false,
    "isMember": true,
    "staffUserId": null,
    "memberId": 1001,
    "deptId": 101,
    "userId": "M1001"
  },
  "requestId": "req_20260707_090020_001"
}
```

## 未登录响应

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "valid": false
  },
  "requestId": null
}
```
