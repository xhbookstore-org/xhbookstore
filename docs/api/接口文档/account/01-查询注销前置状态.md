# 查询注销前置状态

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `GET /api/mp/v1/account/cancel-eligibility` |
| 认证 | Bearer Token |
| 对应代码 | `AccountController.cancelEligibility()` |

## 入参

无。

## 完整请求报文

```http
GET /api/mp/v1/account/cancel-eligibility HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <accessToken>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data.canCancel | boolean | 是否允许注销 |
| data.blockedReason | string | 不允许注销原因 |
| data.hasUnreturnedBooks | boolean | 是否有未还图书 |
| data.isStaffActive | boolean | 是否仍为在职员工 |
| data.currentPoints | int | 当前积分 |

## 完整响应报文

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "canCancel": true,
    "blockedReason": null,
    "hasUnreturnedBooks": false,
    "isStaffActive": false,
    "currentPoints": 100
  },
  "requestId": "req_20260705_110050_001"
}
```
