# 查询注销前置状态

## 基本信息

| 项目 | 内容 |
|---|---|
| 接口 | `GET /api/mp/v1/account/cancel-eligibility` |
| 认证 | 用户 Bearer Token |
| 对应代码 | `AccountController.cancelEligibility()` |

## 入参

无。

## 完整请求报文

```http
GET /api/mp/v1/account/cancel-eligibility HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <userToken>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data.canCancel | boolean | 是否允许注销 |
| data.blockedReason | string | 不允许注销原因，允许注销时为 `null` |
| data.hasUnreturnedBooks | boolean | 是否存在未归还图书 |
| data.isStaffActive | boolean | 当前账号是否仍有关联的在职员工身份 |
| data.currentPoints | int | 当前真实积分 |

## 完整响应报文

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "canCancel": true,
    "blockedReason": null,
    "hasUnreturnedBooks": false,
    "isStaffActive": false,
    "currentPoints": 100
  },
  "requestId": "req_20260705_110050_001"
}
```

## 业务规则

- 必须是有效会员身份。
- 有未归还图书时不允许注销。
- 当前账号仍是有效员工身份时不允许注销。
- `currentPoints` 返回会员当前真实积分，用于前端注销提示展示。
