# 小程序接口文档索引

> 小程序接口基础前缀：`/api/mp/v1`  
> 认证方式：`Authorization: Bearer <accessToken>`

## 分组目录

| 分组 | 目录 | 说明 |
|---|---|---|
| 认证 | [auth](auth) | 微信手机号登录、刷新 Token、校验登录态、退出登录 |
| 用户端 | [user](user) | 用户首页、动态会员码、借阅记录、积分记录 |
| 员工端 | [staff](staff) | 员工首页、扫码会员、借阅、还书、积分、开卡 |
| 账号 | [account](account) | 注销前置状态、注销账号 |
| 文件 | [files](files) | 图书附件图片上传 |

## 当前边界

- 用户端只生成动态会员码，不直接办理开卡和退卡。
- 员工端扫码开卡统一使用 `memberCodeToken` 防刷、防伪造。
- 员工端支付暂按员工录入金额处理，提交即视为支付成功。
- 退卡只在 admin 后台模块办理，小程序端不提供退卡接口。
