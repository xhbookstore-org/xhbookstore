# 查询本人积分记录

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `GET /api/mp/v1/user/points-records` |
| 认证 | 用户 Bearer Token |
| 对应代码 | `UserController.myPointsRecords()` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| pageNo | query | int | 否 | 页码，默认 1，必须大于 0 |
| pageSize | query | int | 否 | 每页条数，默认 20，范围 1-100 |

## 完整请求报文

```http
GET /api/mp/v1/user/points-records?pageNo=1&pageSize=20 HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <userToken>
```

## 业务规则

- 会员 ID 从登录 Token 中解析，只返回当前会员本人的积分流水。
- 分页、总数统计和年度获得积分均由 SQL 查询完成，不进行全量内存分页。
- 积分流水按创建时间倒序返回。

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data.memberDisplay | string | 脱敏会员手机号 |
| data.currentPoints | int | 当前积分 |
| data.yearEarnedPoints | int | 年度获得积分 |
| data.page.list | array | 积分记录 |
| data.page.list[].pointsRecordId | string | 积分记录 ID/订单号 |
| data.page.list[].direction | string | `add=增加`，`deduct=扣减` |
| data.page.list[].pointsDelta | int | 积分变动值 |
| data.page.list[].beforePoints | int | 操作前积分 |
| data.page.list[].afterPoints | int | 操作后积分 |
| data.page.list[].operatedAt | long | 操作时间，毫秒时间戳 |
