# 查询积分详情

## 基本信息

| 项目 | 内容 |
|---|---|
| 接口 | `GET /api/mp/v1/user/points-records/{pointsRecordId}` |
| 认证 | 用户 Bearer Token |
| 对应代码 | `UserController.pointsDetail()` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| pointsRecordId | path | long/string | 是 | 积分记录 ID 或积分订单号，按当前后端支持格式传入 |

## 完整请求报文

```http
GET /api/mp/v1/user/points-records/9001 HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <userToken>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data.pointsRecordId | string | 积分记录 ID/订单号 |
| data.memberId | long | 会员 ID |
| data.reasonName | string | 积分事项名称 |
| data.direction | int/string | 积分方向，增加或消耗 |
| data.pointsDelta | int | 本次积分变动值 |
| data.beforePoints | int | 变动前积分 |
| data.afterPoints | int | 变动后积分 |
| data.operatedAt | string | 操作时间 |
| data.operationDevice | string | 操作来源或设备 |

## 完整响应报文

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "pointsRecordId": "9001",
    "memberId": 1001,
    "reasonName": "活动赠送",
    "direction": "IN",
    "pointsDelta": 10,
    "beforePoints": 120,
    "afterPoints": 130,
    "operatedAt": "2026-07-05 11:10:00",
    "operationDevice": "STAFF_ADMIN"
  },
  "requestId": "req_20260705_111030_001"
}
```

## 业务规则

- 当前登录会员只能查询自己的积分记录详情。
- 如果积分记录不属于当前登录会员，接口返回无权限或未找到。
