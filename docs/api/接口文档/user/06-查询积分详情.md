# 查询积分详情

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `GET /api/mp/v1/user/points-records/{pointsRecordId}` |
| 认证 | 用户 Bearer Token |
| 对应代码 | `UserController.pointsDetail()` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| pointsRecordId | path | string | 是 | 积分记录 ID/订单号 |

## 完整请求报文

```http
GET /api/mp/v1/user/points-records/IN202607050001 HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <userToken>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data.pointsRecordId | string | 积分记录 ID/订单号 |

## 完整响应报文

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "pointsRecordId": "IN202607050001"
  },
  "requestId": "req_20260705_111030_001"
}
```

## 备注

- 当前实现只返回 `pointsRecordId`，后续可补充完整积分流水详情。
