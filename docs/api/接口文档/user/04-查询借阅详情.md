# 查询借阅详情

## 基本信息

| 项目 | 内容 |
|---|---|
| 接口 | `GET /api/mp/v1/user/borrows/{detailId}` |
| 认证 | 用户 Bearer Token |
| 对应代码 | `UserController.borrowDetail()` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| detailId | path | long | 是 | 借阅明细 ID，不是借阅单号 |

## 完整请求报文

```http
GET /api/mp/v1/user/borrows/3001 HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <userToken>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data.order | object | 父借阅单信息 |
| data.detail | object | 当前借阅明细原始信息 |
| data.item | object | 当前借阅明细的前端展示字段 |
| data.item.detailId | long | 借阅明细 ID |
| data.item.orderNo | string | 借阅单号 |
| data.item.bookCode | string | 借阅时录入的图书编号 |
| data.item.bookName | string | 图书名称 |
| data.item.borrowQty | int | 固定为 1 |
| data.item.returnedQty | int | 已还数量 |
| data.item.purchaseQty | int | 借转购数量 |
| data.item.remainingQty | int | 当前可还数量 |
| data.returns | array | 当前明细关联的还书记录 |
| data.returns[].returnOrderNo | string | 还书单号 |
| data.returns[].borrowDetailId | long | 借阅明细 ID |
| data.returns[].returnQty | int | 本次归还数量 |
| data.returns[].returnType | int | 归还类型 |
| data.returns[].returnTime | string | 归还时间 |
| data.images | array | 当前逐册明细图片，最多 3 张 |

## 业务规则

- `detailId` 必须是数字。
- 接口会校验当前 Token 会员 ID 与借阅明细 `memberId` 一致，禁止越权查看其他会员借阅详情。
- 未找到明细返回 `404 / Borrow detail not found`。
