# 查询本人会员卡

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `GET /api/mp/v1/user/member-cards` |
| 认证 | 用户 Bearer Token |
| 对应代码 | `UserController.myMemberCards()` |

## 入参

无。会员 ID 从 Token 解析。

## 完整请求报文

```http
GET /api/mp/v1/user/member-cards HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <userToken>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data.activeCard | object | 当前生效卡 |
| data.pendingCards | array | 待生效卡 |
| data.cards | array | 全部卡片 |
