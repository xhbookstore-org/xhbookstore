# 用户首页

## 基本信息

| 项目 | 内容 |
|---|---|
| 接口 | `GET /api/mp/v1/user/home` |
| 认证 | 用户 Bearer Token |
| 对应代码 | `UserController.home()` |

## 入参

无。

## 完整请求报文

```http
GET /api/mp/v1/user/home HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <userToken>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data.phoneDisplay | string | 脱敏手机号 |
| data.member.memberId | int | 会员 ID |
| data.member.memberNo | string | 会员编号 |
| data.member.memberName | string | 会员姓名 |
| data.member.phoneDisplay | string | 脱敏手机号 |
| data.member.card | object | 当前生效会员卡概要 |
| data.member.currentPoints | int | 当前积分 |
| data.member.currentBorrowingCount | int | 当前在借数量，按借阅明细剩余数量汇总：`borrowQty - returnedQty - purchaseQty` |
| data.member.yearBorrowCount | int | 本年度借阅数量，按本年度借阅明细借出数量汇总 |

## 业务规则

- 只返回当前登录会员本人的首页数据。
- 当前在借数量、年度借阅数量均查询真实借阅明细数据，不再返回固定占位值。
