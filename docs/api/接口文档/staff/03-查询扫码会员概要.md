# 查询扫码会员概要

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `GET /api/mp/v1/staff/members/{memberId}/overview` |
| 认证 | 员工 Bearer Token |
| 对应代码 | `StaffController.memberOverview()` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| memberId | path | string | 是 | 会员 ID |

## 完整请求报文

```http
GET /api/mp/v1/staff/members/1001/overview HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <staffToken>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data.member | object | 会员概要 |
| data.member.memberId | string | 会员 ID |
| data.member.memberNo | string | 会员编号 |
| data.member.memberName | string | 会员姓名 |
| data.member.phoneDisplay | string | 脱敏手机号 |
| data.member.currentPoints | int | 当前积分 |
| data.member.currentBorrowingCount | int | 当前在借数量 |
| data.member.card | object | 当前会员卡概要 |
| data.availability | object | 可操作状态 |
