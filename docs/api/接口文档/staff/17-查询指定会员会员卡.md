# 查询指定会员会员卡

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `GET /api/mp/v1/staff/members/{memberId}/member-cards` |
| 认证 | 员工 Bearer Token |
| 对应代码 | `StaffController.memberCards()` |
| 使用场景 | 员工扫描会员码后，进入会员详情查看该会员的会员卡 |

## 入参

请求头：

```http
Authorization: Bearer <员工 accessToken>
```

路径参数：

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| memberId | int | 是 | 扫码获得的会员 ID |

## 出参

与 `GET /api/mp/v1/user/member-cards` 对齐：

| 字段 | 类型 | 说明 |
|---|---|---|
| data.activeCard | object/null | 当前生效会员卡，状态为 `1` |
| data.pendingCards | array | 待生效会员卡，状态为 `0` |
| data.cards | array | 可见会员卡，包含生效卡和待生效卡 |
| data.hasActiveCard | boolean | 是否存在生效会员卡 |

卡片对象字段包括：`id`、`memberId`、`memberNo`、`cardTypeId`、`cardTypeName`、`validDays`、`saleAmount`、`saleOrderNo`、`status`、`paidAt`、`effectiveAt`、`expiredAt`、`remark`。

## 示例

```http
GET /api/mp/v1/staff/members/1001/member-cards HTTP/1.1
Authorization: Bearer <staffToken>
```

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "activeCard": {
      "id": 9001,
      "cardTypeName": "尊享会员",
      "status": 1,
      "paidAt": "2026-07-05 10:00:00",
      "effectiveAt": "2026-07-05 10:00:00",
      "expiredAt": "2027-07-05 23:59:59"
    },
    "pendingCards": [],
    "cards": [],
    "hasActiveCard": true
  }
}
```

## 异常

- 员工 Token 缺失、过期或无员工身份：返回无员工权限；
- `memberId` 非数字：参数错误；
- 会员不存在：返回会员不存在。
