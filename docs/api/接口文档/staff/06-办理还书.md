# 办理还书

> **接口路径**: `POST /api/mp/v1/staff/borrow-returns`
> **认证要求**: 是（Bearer Token）
> **Content-Type**: application/json
> **对应代码**: `StaffController.java`

## 接口说明

单本或批量归还图书。支持三种归还状态：normal(正常)/damaged(损坏)/lost(丢失)。部分还书是后端内部操作能力，订单级"部分归还"不作为前端列表状态暴露。

## 请求参数

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| borrowOrderNo | String | 是 | 借书单号（DY开头） |
| returnItems | Array | 是 | 要还的图书列表 |
| returnItems[].borrowDetailId | Long | 是 | 借阅明细ID |
| returnItems[].returnStatus | String | 是 | normal / damaged / lost |

## 请求报文

```json
{
  "borrowOrderNo": "DY20260701001",
  "returnItems": [
    {"borrowDetailId": 1001, "returnStatus": "normal"},
    {"borrowDetailId": 1002, "returnStatus": "damaged"}
  ]
}
```

## 响应报文

```json
{
  "code": 0,
  "data": {
    "success": true,
    "returnedCount": 2,
    "totalCount": 2
  }
}
```
