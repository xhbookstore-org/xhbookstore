# 办理还书

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `POST /api/mp/v1/staff/borrow-returns` |
| 认证 | 员工 Bearer Token |
| Content-Type | `application/json;charset=UTF-8` |
| 对应代码 | `StaffController.returnBooks()` |

## 入参

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| returnItems | array | 是 | 还书明细 |
| returnItems[].borrowDetailId | long | 是 | 借阅明细 ID |
| returnItems[].returnQty | int | 是 | 本次归还数量 |
| returnItems[].returnType | int | 否 | 还书类型，默认 `1` |
| returnItems[].remark | string | 否 | 备注 |
| borrowDetailIds | array | 否 | 兼容旧入参；传明细 ID 后默认归还全部可还数量 |

## 完整请求报文

```http
POST /api/mp/v1/staff/borrow-returns HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <staffToken>
Content-Type: application/json;charset=UTF-8

{
  "returnItems": [
    {
      "borrowDetailId": 3001,
      "returnQty": 1,
      "returnType": 1,
      "remark": "正常归还"
    }
  ]
}
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data.returnOrderNos | array | 生成的还书单号列表 |
| data.totalReturned | int | 本次归还总数量 |
| data.traceId | string | 业务追踪 ID |

## 完整响应报文

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "returnOrderNos": ["RT202607050001"],
    "totalReturned": 1,
    "traceId": "ret_20260705_001"
  },
  "requestId": "req_20260705_112040_001"
}
```

## 业务说明

- 按 `borrowDetailId` 直查借阅明细，校验可还数量。
- 同一次请求可跨借阅单，后端会按借阅单分组归还。
- `returnQty` 必须大于 0 且不能超过剩余可还数量。
