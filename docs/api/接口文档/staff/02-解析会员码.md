# 解析会员码

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `POST /api/mp/v1/staff/member-code/scan` |
| 认证 | 员工 Bearer Token |
| Content-Type | `application/json;charset=UTF-8` |
| 对应代码 | `StaffController.scanMemberCode()` |

## 接口说明

员工扫描用户出示的二维码后，将二维码内容作为 `memberCodeToken` 提交给后端。后端校验 token 有效性、场景和会员状态，返回会员概要及会员卡列表。

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| memberCodeToken | body | string | 是 | 用户端生成的动态会员码 token |
| scanResult | body | string | 否 | 兼容旧字段；新业务统一使用 `memberCodeToken` |

## 完整请求报文

```http
POST /api/mp/v1/staff/member-code/scan HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <staffToken>
Content-Type: application/json;charset=UTF-8

{
  "memberCodeToken": "MCT_20260705103000_8f7c6d"
}
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| code | int | `0=成功` |
| message | string | 提示信息 |
| data.memberId | string | 会员 ID |
| data.memberNo | string | 会员编号 |
| data.memberName | string | 会员姓名 |
| data.phoneDisplay | string | 脱敏手机号 |
| data.cards | array | 会员卡列表 |
| requestId | string | 请求追踪 ID |

## 异常响应

```json
{
  "code": 400,
  "message": "memberCodeToken is required",
  "data": null,
  "requestId": "req_20260705_103010_002"
}
```

```json
{
  "code": 400,
  "message": "memberCodeToken expired",
  "data": null,
  "requestId": "req_20260705_103010_003"
}
```

## 业务说明

- 解析时只校验 token，不消费 token；真正开卡时会消费 token。
- 扫码成功后会刷新会员卡状态，用于展示最新的生效卡、待生效卡、过期卡和已退卡。
