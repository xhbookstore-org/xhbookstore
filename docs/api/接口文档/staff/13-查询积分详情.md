# 查询积分详情

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `GET /api/mp/v1/staff/points-records/{pointsRecordId}` |
| 认证 | 员工 Bearer Token |
| 对应代码 | `StaffController.pointsDetail()` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| pointsRecordId | path | string | 是 | 积分记录 ID |

## 完整请求报文

```http
GET /api/mp/v1/staff/points-records/IN202607050001 HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <staffToken>
```

## 出参

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "pointsRecordId": "IN202607050001"
  },
  "requestId": "req_20260705_112110_001"
}
```

## 备注

- 当前实现只返回积分记录 ID，后续可补充完整详情。
