# 查询全市积分列表

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `GET /api/mp/v1/staff/points-records` |
| 认证 | 员工 Bearer Token |
| 对应代码 | `StaffController.pointsRecordsList()` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| phone | query | string | 否 | 会员手机号，当前代码未使用 |
| memberId | query | string | 否 | 会员 ID |
| direction | query | string | 否 | 积分方向，当前代码未使用 |
| pageNo | query | int | 否 | 页码 |
| pageSize | query | int | 否 | 每页条数 |

## 完整请求报文

```http
GET /api/mp/v1/staff/points-records?memberId=1001&pageNo=1&pageSize=20 HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <staffToken>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data.page.list | array | 积分记录 |
| data.page.list[].pointsRecordId | string | 积分记录 ID |
| data.page.list[].direction | string | `add` 或 `deduct` |
| data.page.list[].pointsDelta | int | 积分变动 |
