# 查询员工侧借阅详情

> **接口路径**: `GET /api/mp/v1/staff/borrows/{borrowId}`
> **认证要求**: 是（Bearer Token）
> **对应代码**: `StaffController.java`

## 接口说明

按借书单号(DY开头)查询，返回该单下所有单本借阅项（扁平列表），每本只体现该本的在借/已还状态。

## 路径参数

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| borrowId | String | 是 | 借书单号（DY开头） |

## 响应参数

| 字段 | 类型 | 说明 |
|------|------|------|
| orderNo | String | 借书单号 |
| borrowTime | Long | 借书时间 |
| items[].detailId | Long | 借阅明细ID |
| items[].bookId | Long | 图书ID |
| items[].bookName | String | 书名 |
| items[].borrowStatus | Integer | 该本状态：0=在借 1=已还 |
| items[].borrowTime | Long | 借出时间 |
| items[].returnAllTime | Long | 还回时间 |

## 成功响应

```json
{
  "code": 0,
  "data": {
    "orderNo": "DY20260701001",
    "borrowTime": 1782900000000,
    "items": [
      {"detailId": 1001, "bookId": 1, "bookName": "JavaScript高级程序设计", "borrowStatus": 0, "borrowTime": 1782900000000, "returnAllTime": null},
      {"detailId": 1002, "bookId": 2, "bookName": "深入理解Java虚拟机", "borrowStatus": 1, "borrowTime": 1782900000000, "returnAllTime": 1782950000000}
    ]
  }
}
```
