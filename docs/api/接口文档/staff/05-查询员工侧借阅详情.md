# 查询员工侧借阅详情

## 基本信息

| 项目 | 内容 |
|---|---|
| 接口 | `GET /api/mp/v1/staff/borrows/{detailId}` |
| 认证 | 员工 Bearer Token |
| 对应代码 | `StaffController.borrowDetail()` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| detailId | path | long | 是 | 借阅明细 ID，不是借阅单号 |

## 完整请求报文

```http
GET /api/mp/v1/staff/borrows/3001 HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <staffToken>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data.orderNo | string | 借阅单号 |
| data.borrowTime | long | 当前明细借出时间 |
| data.order | object | 父借阅单信息 |
| data.item | object | 当前借阅明细的平铺展示字段 |
| data.item.detailId | long | 借阅明细 ID |
| data.item.borrowDetailId | long | 借阅明细 ID，兼容字段 |
| data.item.memberId | int | 会员 ID |
| data.item.bookId | long | 图书 ID |
| data.item.bookName | string | 图书名称 |
| data.item.borrowQty | int | 借出数量 |
| data.item.returnedQty | int | 已还数量 |
| data.item.purchaseQty | int | 借转购数量 |
| data.item.remainingQty | int | 当前可还数量 |
| data.returns | array | 当前明细关联的还书记录 |

## 业务规则

- 该接口用于员工查看某个用户的一本书借阅详情。
- `detailId` 必须是数字，未找到明细返回 `404 / Borrow detail not found`。
- 返回的 `returns` 只包含当前 `detailId` 对应的还书记录，不返回同一借阅单下其他图书的还书记录。
