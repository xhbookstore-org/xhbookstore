# 查询指定会员借阅记录

> **接口路径**: `GET /api/mp/v1/staff/members/{memberId}/borrows`
> **认证要求**: 是（Bearer Token）
> **对应代码**: `StaffController.java`

## 接口说明

单本借阅项扁平返回。mode=current返回未归还项，mode=all返回全部。不按借书单分组，不暴露订单级"部分归还"状态。

## 路径参数

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| memberId | String | 是 | 会员ID |

## 请求参数

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| mode | String | 否 | current=未归还 / all=全部（默认） |
| pageNo | int | 否 | 页码，默认1 |
| pageSize | int | 否 | 每页条数，默认20 |

> 已取消 `status` 筛选参数。

## curl 测试

```bash
# 查询在借记录
curl 'http://localhost:8091/api/mp/v1/staff/members/10/borrows?mode=current' \
  -H "Authorization: Bearer <staffToken>"
```

## 响应参数

| 字段 | 类型 | 说明 |
|------|------|------|
| member.memberId | String | 会员ID |
| member.memberNo | String | 会员卡号 |
| member.memberName | String | 姓名 |
| page.list[].detailId | Long | 借阅明细ID |
| page.list[].orderNo | String | 借书单号 |
| page.list[].bookId | Long | 图书ID |
| page.list[].bookName | String | 书名 |
| page.list[].borrowStatus | Integer | 该本状态：0=在借 1=已还 |
| page.list[].borrowTime | Long | 借出时间 |
