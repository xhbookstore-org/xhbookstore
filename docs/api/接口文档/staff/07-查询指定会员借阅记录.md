# 查询指定会员借阅记录

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `GET /api/mp/v1/staff/members/{memberId}/borrows` |
| 认证 | 员工 Bearer Token |
| 对应代码 | `StaffController.memberBorrows()` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| memberId | path | string | 是 | 会员 ID |
| mode | query | string | 否 | `current` 只查当前在借；其他查全部 |
| status | query | string | 否 | 状态筛选，当前代码未使用 |
| pageNo | query | int | 否 | 页码 |
| pageSize | query | int | 否 | 每页条数 |

## 完整请求报文

```http
GET /api/mp/v1/staff/members/1001/borrows?mode=current&pageNo=1&pageSize=20 HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <staffToken>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data.member | object | 会员概要 |
| data.page | object | 借阅明细分页 |
