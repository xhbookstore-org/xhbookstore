# 查询全市借阅列表

> **接口路径**: `GET /api/mp/v1/staff/borrows`
> **认证要求**: 是（Bearer Token）
> **对应代码**: `StaffController.java`

## 接口说明

单本借阅项扁平返回——每一条记录代表一本书的借阅。不按借书单+details[]分组，也不暴露订单级"部分归还"状态。

## 请求参数

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| phone | String | 否 | 手机号筛选 |

## 响应参数

| 字段 | 类型 | 说明 |
|------|------|------|
| detailId | Long | 借阅明细ID |
| orderNo | String | 借书单号（DY开头） |
| memberId | Integer | 会员ID |
| bookId | Long | 图书ID |
| bookName | String | 书名 |
| borrowStatus | Integer | 该本书状态：0=在借 1=已还 |
| borrowTime | Long | 借出时间 |
| returnAllTime | Long | 还回时间 |
