# 查询全市借阅列表

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `GET /api/mp/v1/staff/borrows` |
| 认证 | 员工 Bearer Token |
| 对应代码 | `StaffController.borrowsList()` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| phone | query | string | 否 | 会员手机号 |
| status | query | string | 否 | 借阅状态 |
| pageNo | query | int | 否 | 页码，默认 1 |
| pageSize | query | int | 否 | 每页条数，默认 20 |

## 完整请求报文

```http
GET /api/mp/v1/staff/borrows?pageNo=1&pageSize=20&phone=13800000000&status=borrowing HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <staffToken>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data.page | object | 分页对象 |
| data.page.list | array | 借阅列表 |
| data.page.total | long | 总数 |

## 完整响应报文

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "page": {
      "list": [],
      "pageNo": 1,
      "pageSize": 20,
      "total": 0,
      "hasMore": false
    }
  },
  "requestId": "req_20260705_112020_001"
}
```

## 备注

- 当前实现返回空分页，后续可接入真实全市借阅查询。
