# 查询借阅列表

## 基本信息

| 项目 | 内容 |
|---|---|
| 接口 | `GET /api/mp/v1/staff/borrows` |
| 认证 | 员工 Bearer Token |
| 对应代码 | `StaffController.borrowsList()` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| phone | query | string | 否 | 会员手机号，支持按手机号筛选 |
| status | query | int | 否 | 借阅状态值，对应借阅明细状态 |
| pageNo | query | int | 否 | 页码，默认 `1` |
| pageSize | query | int | 否 | 每页条数，默认 `20` |

## 完整请求报文

```http
GET /api/mp/v1/staff/borrows?pageNo=1&pageSize=20&phone=13800000000&status=1 HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <staffToken>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data.page | object | 分页对象 |
| data.page.list | array | 借阅明细列表，每本书一条记录 |
| data.page.total | long | 总数 |
| data.page.pageNo | int | 当前页码 |
| data.page.pageSize | int | 每页条数 |
| data.page.hasMore | boolean | 是否还有下一页 |
| data.page.list[].detailId | long | 借阅明细 ID |
| data.page.list[].borrowDetailId | long | 借阅明细 ID，兼容字段 |
| data.page.list[].orderNo | string | 借阅单号 |
| data.page.list[].memberId | long | 会员 ID |
| data.page.list[].bookId | long | 图书 ID |
| data.page.list[].bookName | string | 图书名称 |
| data.page.list[].borrowStatus | int | 借阅状态 |
| data.page.list[].borrowQty | int | 借出数量 |
| data.page.list[].returnedQty | int | 已还数量 |
| data.page.list[].purchaseQty | int | 借转购数量 |
| data.page.list[].remainingQty | int | 当前可还数量 |
| data.page.list[].purchaseOrderNo | string | 借转购订单号 |
| data.page.list[].borrowTime | long/string | 借出时间 |
| data.page.list[].returnAllTime | long/string | 全部归还时间 |
| data.page.list[].expectedReturnTime | long/string | 预计归还时间 |
| data.page.list[].remark | string | 备注 |

## 业务规则

- 返回真实借阅明细列表，不再返回固定空分页。
- 列表按“每本书一条借阅明细”返回，而不是按借阅单分组返回。
- 查询结果受当前员工数据权限限制，只返回员工可见部门/门店范围内的会员借阅数据。
