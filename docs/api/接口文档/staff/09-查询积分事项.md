# 查询积分事项

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `GET /api/mp/v1/staff/points-reasons` |
| 认证 | 员工 Bearer Token |
| 对应代码 | `StaffController.pointsReasons()` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| direction | query | string | 是 | `add=增加`，`deduct=扣减` |
| memberId | query | string | 否 | 会员 ID |

## 完整请求报文

```http
GET /api/mp/v1/staff/points-reasons?direction=add&memberId=1001 HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <staffToken>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data.list | array | 积分事项 |
| data.list[].reasonId | string | 事项 ID |
| data.list[].reasonName | string | 事项名称 |
| data.list[].enabled | boolean | 是否启用 |
| data.list[].defaultPoints | int | 默认积分 |
| data.maxPoints | int | 最大可操作积分 |

## 完整响应报文

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "list": [
      {
        "reasonId": "1",
        "reasonName": "activity_bonus",
        "enabled": true,
        "defaultPoints": 50
      }
    ],
    "maxPoints": 99999
  },
  "requestId": "req_20260705_112070_001"
}
```
