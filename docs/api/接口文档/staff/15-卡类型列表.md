# 卡类型列表

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `GET /api/mp/v1/staff/card-types` |
| 认证 | 员工 Bearer Token |
| 对应代码 | `StaffController.cardTypes()` |

## 入参

无。

## 完整请求报文

```http
GET /api/mp/v1/staff/card-types HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <staffToken>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data.list | array | 卡类型列表 |
| data.list[].id | int | 卡类型 ID |
| data.list[].typeName | string | 卡类型名称 |
| data.list[].price | decimal | 售价 |
| data.list[].validDays | int | 有效天数 |
| data.list[].borrowLimit | int | 借阅上限 |
| data.list[].discount | decimal | 折扣 |
| data.list[].isRenewal | boolean | 是否续费类型 |
| data.list[].description | string | 描述 |
| data.list[].sort | int | 排序 |
| data.list[].status | int | 状态 |
