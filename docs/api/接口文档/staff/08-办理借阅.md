# 办理借阅

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `POST /api/mp/v1/staff/members/{memberId}/borrows` |
| 认证 | 员工 Bearer Token |
| Content-Type | `application/json;charset=UTF-8` |
| 对应代码 | `StaffController.borrow()` |

## 入参

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| memberId | path | string | 是 | 会员 ID |
| books | array | 是 | 借阅图书 |
| books[].bookId | long | 是 | 图书 ID |
| books[].bookName | string | 否 | 图书名称 |
| books[].borrowQty | int | 是 | 借阅数量 |
| remark | string | 否 | 备注 |
| imageUrls | array | 否 | 附件图片 URL |

## 完整请求报文

```http
POST /api/mp/v1/staff/members/1001/borrows HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <staffToken>
Content-Type: application/json;charset=UTF-8

{
  "books": [
    {
      "bookId": 2001,
      "bookName": "活着",
      "borrowQty": 2
    }
  ],
  "remark": "门店借阅",
  "imageUrls": [
    "https://cos.example.com/bookstore/1001/book.jpg"
  ]
}
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data.orderNo | string | 借阅单号 |
| data.borrowOrderId | long | 借阅单 ID |
| data.totalBookCount | int | 借阅总数量 |

## 完整响应报文

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "orderNo": "BR202607050001",
    "borrowOrderId": 5001,
    "totalBookCount": 2
  },
  "requestId": "req_20260705_112060_001"
}
```
