# 增加积分

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `POST /api/mp/v1/staff/members/{memberId}/points/add` |
| 认证 | 员工 Bearer Token |
| Content-Type | `application/json;charset=UTF-8` |
| 对应代码 | `StaffController.addPoints()` |

## 入参

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| memberId | path | string | 是 | 会员 ID |
| points | int | 是 | 增加积分，必须大于 0 |
| remark | string | 否 | 备注/原因 |

## 完整请求报文

```http
POST /api/mp/v1/staff/members/1001/points/add HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <staffToken>
Content-Type: application/json;charset=UTF-8

{
  "points": 20,
  "remark": "活动赠送"
}
```

{
  "code": 0,
  "message": "操作成功",
