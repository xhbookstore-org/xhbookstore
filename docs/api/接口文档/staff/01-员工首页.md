# 员工首页

## 基本信息

| 项目 | 内容 |
|---|---|
| 接口 | `GET /api/mp/v1/staff/home` |
| 认证 | 员工 Bearer Token |
| 对应代码 | `StaffController.home()` |

## 入参

无。

## 完整请求报文

```http
GET /api/mp/v1/staff/home HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <staffToken>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data.storeName | string | 当前员工所属部门/门店名称 |
| data.todayStoreBorrowCount | int | 今日当前部门/门店借阅单数量 |
| data.todayStaffBorrowCount | int | 当前员工今日办理借阅单数量 |

## 完整响应报文

```json
{
  "code": 0,
  "message": "操作成功",
  "data": {
    "storeName": "新华书店中心店",
    "todayStoreBorrowCount": 12,
    "todayStaffBorrowCount": 3
  },
  "requestId": "req_20260705_112000_001"
}
```

## 业务规则

- `storeName` 取当前员工部门名称。
- `todayStoreBorrowCount` 按当前员工部门统计当天借阅单数量。
- `todayStaffBorrowCount` 按当前员工 ID 统计当天本人办理借阅单数量。
