# 公共接口

> 认证相关接口

---

## 登录

| 方法 | 路径 | 认证 | 说明 |
|------|------|------|------|
| GET | `/captchaImage` | 否 | 获取验证码图片 |
| POST | `/login` | 否 | 用户登录 |

### POST /login 请求参数

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| username | string | 是 | 登录账号 |
| password | string | 是 | 密码 |
| code | string | 是 | 验证码 |
| uuid | string | 是 | 验证码唯一标识 |

### POST /login 返回

```json
{ "code": 0, "msg": "操作成功", "data": { "token": "xxx" } }
```

## 统一响应格式

所有接口统一返回：

```json
{
  "code": 0,
  "msg": "操作成功",
  "data": {},
  "rows": [],
  "total": 0
}
```

### 分页格式

分页接口使用 `rows` + `total`：
```json
{
  "code": 0,
  "msg": "查询成功",
  "rows": [...],
  "total": 100
}
```

### 异常码

| code | 说明 |
|------|------|
| 0 | 操作成功 |
| 400 | 参数错误 |
| 401 | 未登录 / 登录过期 |
| 403 | 无权限 |
| 500 | 服务端错误 |
