# 查询会员卡展示视图

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `GET /member/card/member/{memberId}` |
| 权限 | `member:card:list` |
| 返回类型 | `AjaxResult` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| memberId | path | int | 是 | 会员 ID |

## 完整请求报文

```http
GET /member/card/member/1001 HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <token>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data.activeCard | object | 当前生效会员卡；没有则为 `null` |
| data.pendingCards | array | 待生效会员卡列表 |
| data.cards | array | 全部会员卡列表 |

## 业务说明

- 该接口会读取会员卡队列视图，用于会员列表右侧退卡弹窗和会员绑卡情况展示。
- 会员卡状态刷新由业务服务处理，不依赖定时任务。
