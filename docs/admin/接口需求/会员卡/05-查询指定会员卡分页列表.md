# 查询指定会员卡分页列表

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `GET /member/card/member/{memberId}/list` |
| 权限 | `member:card:list` |
| 返回类型 | `TableDataInfo` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| memberId | path | int | 是 | 会员 ID |
| pageNum | query | int | 否 | 页码 |
| pageSize | query | int | 否 | 每页条数 |

## 完整请求报文

```http
GET /member/card/member/1001/list?pageNum=1&pageSize=10 HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <token>
```

## 出参

返回字段同 [查询会员卡记录列表](01-查询会员卡记录列表.md) 的 `rows[]`。

## 完整响应报文

```json
{
  "code": 200,
  "msg": "查询成功",
  "total": 2,
  "rows": [
    {
      "id": 9001,
      "memberId": 1001,
      "memberNo": "65000000001",
      "cardTypeName": "年卡",
      "status": 1,
      "paidAt": "2026-07-05 10:00:00",
      "effectiveAt": "2026-07-05 10:00:00",
      "expiredAt": "2027-07-05 23:59:59"
    }
  ]
}
```

## 业务说明

- 查询前会调用 `refreshMemberCardStatus(memberId, 当前用户, 当前用户, "ADMIN")`。
- 如果当前生效卡已过期，会在查询时推进下一张待生效卡。
