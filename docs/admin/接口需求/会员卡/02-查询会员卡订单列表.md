# 查询会员卡订单列表

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `GET /member/card/order/list` |
| 权限 | `member:cardOrder:list`、`member:card:list` 或 `member:card:query` 任一权限 |
| 数据权限 | `@DataScope(deptAlias = "o")` |
| 返回类型 | `TableDataInfo` |
| 默认排序 | `pay_time desc, created_at desc, id desc` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| pageNum | query | int | 否 | 页码 |
| pageSize | query | int | 否 | 每页条数 |
| orderNo | query | string | 否 | 售卡订单号，模糊查询 |
| memberCardId | query | long | 否 | 会员卡 ID |
| memberId | query | int | 否 | 会员 ID |
| memberNo | query | string | 否 | 会员编号，模糊查询 |
| memberNameLike | query | string | 否 | 会员姓名，模糊查询 |
| memberPhone | query | string | 否 | 手机号，模糊查询 |
| cardTypeId | query | int | 否 | 卡类型 ID |
| cardTypeName | query | string | 否 | 卡类型名称，模糊查询 |
| paymentType | query | string | 否 | 支付方式，如 `STAFF_INPUT` |
| orderStatus | query | int | 否 | 订单状态，当前成功订单通常为 `1` |
| deptId | query | long | 否 | 门店 ID |
| createStaffName | query | string | 否 | 开单员工，模糊查询 |
| beginPayTime | query | string | 否 | 支付开始时间 |
| endPayTime | query | string | 否 | 支付结束时间 |
| beginCreatedAt | query | string | 否 | 创建开始时间 |
| endCreatedAt | query | string | 否 | 创建结束时间 |

## 完整请求报文

```http
GET /member/card/order/list?pageNum=1&pageSize=10&memberNo=65000000001&orderStatus=1&beginPayTime=2026-07-01%2000:00:00&endPayTime=2026-07-31%2023:59:59 HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <token>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| rows[].id | long | 订单 ID |
| rows[].orderNo | string | 售卡订单号 |
| rows[].memberCardId | long | 关联会员卡 ID |
| rows[].memberId | int | 会员 ID |
| rows[].memberNo | string | 会员编号 |
| rows[].memberName | string | 会员姓名 |
| rows[].cardTypeName | string | 卡类型 |
| rows[].validDays | int | 有效天数 |
| rows[].receivableAmount | decimal | 应收金额 |
| rows[].paidAmount | decimal | 实收金额 |
| rows[].paymentType | string | 支付方式 |
| rows[].orderStatus | int | 订单状态 |
| rows[].payTime | string | 支付时间 |
| rows[].deptName | string | 门店 |
| rows[].createStaffName | string | 开单员工 |

## 业务说明

- 开卡/续卡成功必须生成售卡订单。
- 订单与会员卡通过 `memberCardId` 闭环关联。
