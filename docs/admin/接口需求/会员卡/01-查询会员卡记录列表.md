# 查询会员卡记录列表

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `GET /member/card/list` |
| 权限 | `member:card:list` |
| 数据权限 | `@DataScope(deptAlias = "c")` |
| 返回类型 | `TableDataInfo` |
| 默认排序 | `field(status, 1, 0, 3, 2), coalesce(paid_at, effective_at, created_at) desc, id desc` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| pageNum | query | int | 否 | 页码 |
| pageSize | query | int | 否 | 每页条数 |
| memberId | query | int | 否 | 会员 ID |
| memberName | query | string | 否 | 会员姓名，模糊查询 |
| memberPhone | query | string | 否 | 手机号，模糊查询 |
| memberNo | query | string | 否 | 会员编号，模糊查询 |
| cardTypeId | query | int | 否 | 卡类型 ID |
| status | query | int | 否 | `0=待生效`，`1=生效中`，`2=已过期`，`3=已退卡` |
| saleOrderNo | query | string | 否 | 售卡订单号，模糊查询 |
| refundOrderNo | query | string | 否 | 退卡订单号，模糊查询 |
| deptId | query | long | 否 | 门店 ID |
| createStaffName | query | string | 否 | 开卡员工姓名，模糊查询 |
| beginPaidAt | query | string | 否 | 付款开始时间 |
| endPaidAt | query | string | 否 | 付款结束时间 |
| beginEffectiveAt | query | string | 否 | 生效开始时间 |
| endEffectiveAt | query | string | 否 | 生效结束时间 |
| beginExpiredAt | query | string | 否 | 到期开始时间 |
| endExpiredAt | query | string | 否 | 到期结束时间 |

## 完整请求报文

```http
GET /member/card/list?pageNum=1&pageSize=10&memberNo=65000000001&status=1&beginPaidAt=2026-07-01%2000:00:00&endPaidAt=2026-07-31%2023:59:59 HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <token>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| code | int | 状态码 |
| total | long | 总数 |
| rows | array | 会员卡记录 |
| rows[].id | long | 会员卡 ID |
| rows[].memberId | int | 会员 ID |
| rows[].memberNo | string | 会员编号 |
| rows[].memberName | string | 会员姓名 |
| rows[].memberPhone | string | 会员手机号 |
| rows[].cardTypeId | int | 卡类型 ID |
| rows[].cardTypeName | string | 卡类型名称 |
| rows[].validDays | int | 有效天数 |
| rows[].saleAmount | decimal | 售卡金额 |
| rows[].saleOrderNo | string | 售卡订单号 |
| rows[].status | int | 卡状态 |
| rows[].paidAt | string | 付款时间 |
| rows[].effectiveAt | string | 生效时间 |
| rows[].expiredAt | string | 到期时间 |
| rows[].refundedAt | string | 退卡时间 |
| rows[].refundOrderNo | string | 退卡订单号 |
| rows[].deptName | string | 门店名称 |
| rows[].createStaffName | string | 开卡员工 |
