# 查询会员卡业务日志

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `GET /member/card/{memberCardId}/log` |
| 权限 | `member:card:query` |
| 返回类型 | `AjaxResult` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| memberCardId | path | long | 是 | 会员卡 ID |

## 完整请求报文

```http
GET /member/card/9001/log HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <token>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| data[].id | long | 日志 ID |
| data[].memberCardId | long | 会员卡 ID |
| data[].bizType | string | 业务类型，如 `BUY`、`ACTIVATE`、`EXPIRE`、`REFUND` |
| data[].saleOrderNo | string | 售卡订单号 |
| data[].refundOrderNo | string | 退卡订单号 |
| data[].beforeData | string | 操作前快照 |
| data[].afterData | string | 操作后快照 |
| data[].operatorId | string | 操作人 ID |
| data[].operatorName | string | 操作人名称 |
| data[].createdAt | string | 记录时间 |
