# 查询会员退卡订单列表

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `GET /member/card/refund-order/list` |
| 权限 | `member:cardRefund:list`、`member:card:list` 或 `member:card:query` 任一权限 |
| 数据权限 | `@DataScope(deptAlias = "r")` |
| 返回类型 | `TableDataInfo` |
| 默认排序 | `refund_time desc, created_at desc, id desc` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| pageNum | query | int | 否 | 页码 |
| pageSize | query | int | 否 | 每页条数 |
| refundOrderNo | query | string | 否 | 退卡订单号，模糊查询 |
| saleOrderNo | query | string | 否 | 原售卡订单号，模糊查询 |
| memberCardId | query | long | 否 | 会员卡 ID |
| memberId | query | int | 否 | 会员 ID |
| memberNo | query | string | 否 | 会员编号，模糊查询 |
| memberNameLike | query | string | 否 | 会员姓名，模糊查询 |
| memberPhone | query | string | 否 | 手机号，模糊查询 |
| cardTypeId | query | int | 否 | 卡类型 ID |
| cardTypeName | query | string | 否 | 卡类型名称，模糊查询 |
| refundType | query | string | 否 | 退款方式，如 `CASH` |
| refundStatus | query | int | 否 | 退款状态，当前成功退卡通常为 `1` |
| deptId | query | long | 否 | 门店 ID |
| operatorName | query | string | 否 | 退卡操作人，模糊查询 |
| beginRefundTime | query | string | 否 | 退款开始时间 |
| endRefundTime | query | string | 否 | 退款结束时间 |
| beginCreatedAt | query | string | 否 | 创建开始时间 |
| endCreatedAt | query | string | 否 | 创建结束时间 |

## 完整请求报文

```http
GET /member/card/refund-order/list?pageNum=1&pageSize=10&memberNo=65000000001&refundType=CASH&beginRefundTime=2026-07-01%2000:00:00&endRefundTime=2026-07-31%2023:59:59 HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <token>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| rows[].id | long | 退卡订单 ID |
| rows[].refundOrderNo | string | 退卡订单号 |
| rows[].saleOrderNo | string | 原售卡订单号 |
| rows[].memberCardId | long | 会员卡 ID |
| rows[].memberNo | string | 会员编号 |
| rows[].memberName | string | 会员姓名 |
| rows[].cardTypeName | string | 卡类型 |
| rows[].refundAmount | decimal | 退款金额 |
| rows[].refundType | string | 退款方式 |
| rows[].refundStatus | int | 退款状态 |
| rows[].reason | string | 退款原因 |
| rows[].refundTime | string | 退款时间 |
| rows[].operatorName | string | 操作人 |
