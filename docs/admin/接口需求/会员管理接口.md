# 会员管理接口索引

> 后台接口基础前缀：`/member`
> 认证方式：管理后台登录态 + `Authorization: Bearer <token>`
> 通用返回：普通 JSON 接口使用 `AjaxResult`，分页接口使用 `TableDataInfo`，导出接口返回 Excel 文件流。

## 重要约定

- 会员列表、导出接口均受 `@DataScope(deptAlias = "m")` 限制，只能查询当前登录人有权限的数据范围。
- 会员导入使用当前登录人作为操作人；新增会员需要写入来源字段，重复会员编号/会员卡号归为“未导入”。
- 导出接口导出符合查询条件的全量数据，不受分页参数影响。
