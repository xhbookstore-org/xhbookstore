# 会员卡业务接口索引

> 后台接口基础前缀：`/member/card`  
> 认证方式：管理后台登录态 + `Authorization: Bearer <token>`  
> 通用返回：普通 JSON 接口使用 `AjaxResult`，分页接口使用 `TableDataInfo`。

## 重要约定

- 会员卡状态：`0=待生效`，`1=生效中`，`2=已过期`，`3=已退卡`。
- 会员卡列表、订单列表、退卡订单列表均受数据权限限制。
- 会员卡记录导出受数据权限限制，按当前查询条件导出全量数据，不受分页参数限制。
- 退卡以付款时间 `paidAt` 为准，7 日内允许退卡，不以生效时间为准。
- 退卡只能在 admin 后台由有 `member:card:refund` 权限的员工操作，小程序端不提供退卡接口。
