# 编辑会员

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `PUT /member/{id}` |
| 权限 | `member:member:edit` |
| 日志 | `@Log(title = "会员管理", businessType = UPDATE)` |
| 返回类型 | `AjaxResult` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| id | path | int | 是 | 会员主键 ID |
| name | body | string | 否 | 会员姓名 |
| phone | body | string | 否 | 手机号 |
| cardTypeId | body | int | 否 | 当前卡类型 ID |
| levelId | body | int | 否 | 等级 ID |
| deptId | body | long | 否 | 所属门店 |
| validDate | body | string | 否 | 有效期截止日期 |
| status | body | int | 否 | 状态 |
| currentPoints | body | int | 否 | 当前积分 |
| remark | body | string | 否 | 备注 |

## 完整请求报文

```http
PUT /member/1001 HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <token>
Content-Type: application/json

{
  "name": "张三",
  "phone": "13800000000",
  "cardTypeId": 2,
  "validDate": "2027-12-31",
  "remark": "后台维护"
}
```

## 出参

```json
{
  "code": 200,
  "msg": "操作成功"
}
```

## 备注

- 路径中的 `id` 会覆盖 body 中的 `id`。
- 更新时会刷新 `updated_at`。
