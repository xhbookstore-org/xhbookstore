# 查询卡类型下拉

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `GET /member/cardTypes` |
| 权限 | `member:member:query` |
| 返回类型 | `AjaxResult` |

## 入参

无。

## 完整请求报文

```http
GET /member/cardTypes HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <token>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| code | int | 状态码 |
| data | array | 卡类型列表 |
| data[].id | int | 卡类型 ID |
| data[].typeName | string | 卡类型名称 |
| data[].validDays | int | 有效天数 |
| data[].status | int | 状态 |
| data[].isRenewal | int | 是否续费类型 |

## 完整响应报文

```json
{
  "code": 200,
  "msg": "操作成功",
  "data": [
    {
      "id": 1,
      "typeName": "年卡",
      "validDays": 365,
      "status": 0,
      "isRenewal": 0
    }
  ]
}
```
