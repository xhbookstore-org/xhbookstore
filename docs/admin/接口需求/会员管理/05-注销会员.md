# 注销会员

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `DELETE /member/{id}` |
| 权限 | `member:member:remove` |
| 日志 | `@Log(title = "会员管理", businessType = DELETE)` |
| 返回类型 | `AjaxResult` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| id | path | int | 是 | 会员主键 ID |

## 完整请求报文

```http
DELETE /member/1001 HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <token>
```

## 出参

```json
{
  "code": 200,
  "msg": "操作成功"
}
```

## 业务说明

- 当前实现为逻辑注销：`member.status = 1`，并更新 `updated_at`。
- 不物理删除会员数据。
