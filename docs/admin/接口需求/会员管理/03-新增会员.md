# 新增会员

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `POST /member` |
| 权限 | `member:member:add` |
| 日志 | `@Log(title = "会员管理", businessType = INSERT)` |
| 返回类型 | `AjaxResult` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| cardNo | body | string | 否 | 会员编号；为空时由业务层生成 |
| name | body | string | 是 | 会员姓名 |
| phone | body | string | 否 | 手机号 |
| cardTypeId | body | int | 否 | 当前卡类型 ID |
| levelId | body | int | 否 | 等级 ID |
| deptId | body | long | 否 | 所属门店；为空使用当前登录人门店 |
| validDate | body | string | 否 | 有效期截止日期，格式 `yyyy-MM-dd` |
| status | body | int | 否 | 状态，默认按业务层处理 |
| currentPoints | body | int | 否 | 当前积分 |
| remark | body | string | 否 | 备注 |

## 完整请求报文

```http
POST /member HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <token>
Content-Type: application/json

{
  "cardNo": "65000000088",
  "name": "李四",
  "phone": "13900000000",
  "cardTypeId": 1,
  "levelId": 1,
  "deptId": 103,
  "validDate": "2027-06-30",
  "currentPoints": 0,
  "remark": "后台手工新增"
}
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| code | int | 状态码 |
| msg | string | 提示信息 |

## 完整响应报文

```json
{
  "code": 200,
  "msg": "操作成功"
}
```

## 业务说明

- `deptId` 为空时自动取当前登录用户部门。
- 新增会员应写入操作日志。
