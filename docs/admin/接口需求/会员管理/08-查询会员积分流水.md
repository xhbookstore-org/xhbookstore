# 查询会员积分流水

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `GET /member/{id}/points` |
| 权限 | `member:member:query` |
| 返回类型 | `AjaxResult` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| id | path | int | 是 | 会员主键 ID |

## 完整请求报文

```http
GET /member/1001/points HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <token>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| code | int | 状态码 |
| data | array | 积分订单/流水列表 |

## 完整响应报文

```json
{
  "code": 200,
  "msg": "操作成功",
  "data": [
    {
      "id": 5001,
      "memberId": 1001,
      "points": 20,
      "description": "活动赠送",
      "operator": "admin",
      "device": "PC",
      "createdAt": "2026-07-05 10:30:00"
    }
  ]
}
```
