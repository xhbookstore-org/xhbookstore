# 增加会员积分

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `POST /member/{id}/points` |
| 权限 | `member:member:points` |
| 日志 | `@Log(title = "会员积分", businessType = UPDATE)` |
| 返回类型 | `AjaxResult` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| id | path | int | 是 | 会员主键 ID |
| points | body | int | 是 | 增加积分数量，应为正整数 |
| description | body | string | 否 | 积分说明 |

## 完整请求报文

```http
POST /member/1001/points HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <token>
Content-Type: application/json

{
  "points": 20,
  "description": "活动赠送"
}
```

## 出参

```json
{
  "code": 200,
  "msg": "操作成功"
}
```

## 业务说明

- 操作人使用当前登录用户名。
- 设备来源固定传 `PC`。
