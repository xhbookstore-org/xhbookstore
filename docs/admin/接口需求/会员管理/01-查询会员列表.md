# 查询会员列表

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `GET /member/list` |
| 权限 | `member:member:list` |
| 数据权限 | `@DataScope(deptAlias = "m")` |
| 返回类型 | `TableDataInfo` |
| 默认排序 | `coalesce(updated_at, created_at) desc, created_at desc, id desc` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| pageNum | query | int | 否 | 页码，默认 1 |
| pageSize | query | int | 否 | 每页条数，默认 10 |
| status | query | int | 否 | 会员状态，`0=正常`，`1=注销` |
| name | query | string | 否 | 会员姓名，模糊查询 |
| cardNo | query | string | 否 | 会员编号，模糊查询 |
| phone | query | string | 否 | 手机号，模糊查询 |
| deptId | query | long | 否 | 门店 ID |
| params[beginTime] | query | string | 否 | 创建开始日期，格式 `yyyy-MM-dd` |
| params[endTime] | query | string | 否 | 创建结束日期，格式 `yyyy-MM-dd` |

## 完整请求报文

```http
GET /member/list?pageNum=1&pageSize=10&cardNo=65000000001&name=张三&phone=13800000000&status=0&deptId=103&params[beginTime]=2026-07-01&params[endTime]=2026-07-31 HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <token>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| code | int | 状态码，`200` 表示成功 |
| msg | string | 提示信息 |
| total | long | 符合条件的总记录数 |
| rows | array | 会员列表 |
| rows[].id | int | 会员主键 ID |
| rows[].cardNo | string | 会员编号 |
| rows[].name | string | 姓名 |
| rows[].phone | string | 手机号 |
| rows[].cardTypeId | int | 当前会员卡类型 ID |
| rows[].cardTypeName | string | 当前会员卡类型名称 |
| rows[].levelId | int | 会员等级 ID |
| rows[].deptId | long | 所属门店 ID |
| rows[].deptName | string | 所属门店名称 |
| rows[].validDate | string | 当前有效期截止日期，格式 `yyyy-MM-dd` |
| rows[].status | int | 会员状态 |
| rows[].currentPoints | int | 当前积分 |
| rows[].source | string | 会员来源，如 `manual`、`erp_import` |
| rows[].syncErp | int | 是否同步 ERP，`0=否`，`1=是` |
| rows[].createdAt | string | 创建时间 |
| rows[].updatedAt | string | 更新时间 |
