# 查询会员详情

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `GET /member/{id}` |
| 权限 | `member:member:query` |
| 返回类型 | `AjaxResult` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| id | path | int | 是 | 会员主键 ID |

## 完整请求报文

```http
GET /member/1001 HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <token>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| code | int | 状态码 |
| msg | string | 提示信息 |
| member | object | 会员主表信息 |
| ext | object | 会员扩展信息 |
