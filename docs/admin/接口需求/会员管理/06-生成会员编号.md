# 生成会员编号

## 基本信息

| 项 | 内容 |
|---|---|
| 接口 | `GET /member/generateCardNo` |
| 权限 | `member:member:add` |
| 返回类型 | `AjaxResult` |

## 入参

| 字段 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| deptId | query | long | 否 | 门店 ID；为空使用当前登录人门店 |

## 完整请求报文

```http
GET /member/generateCardNo?deptId=103 HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer <token>
```

## 出参

| 字段 | 类型 | 说明 |
|---|---|---|
| code | int | 状态码 |
| msg | string | 固定返回 `ok` |
| data | string | 新会员编号 |

## 完整响应报文

```json
{
  "code": 200,
  "msg": "ok",
  "data": "65000000089"
}
```
