# 系统管理接口

> 基础路径：`/system/**` | 认证方式：Session Cookie

---

## 用户管理

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/system/user/list` | 用户列表（分页） |
| GET | `/system/user/{id}` | 用户详情 |
| POST | `/system/user` | 新增用户 |
| PUT | `/system/user` | 编辑用户 |
| DELETE | `/system/user/{ids}` | 删除用户 |
| PUT | `/system/user/resetPwd` | 重置密码 |
| GET | `/system/user/export` | 导出用户 |
