# 首页会员数据看板接口

## 1. 获取首页会员数据看板

### 基本信息

| 项目 | 内容 |
|---|---|
| 接口 | `GET /dashboard/member/overview` |
| 权限 | admin 后台登录态 |
| 入参 | 无 |
| 数据范围 | 后端按当前登录人部门和角色数据权限自动计算 |
| 数据来源 | Redis 预聚合数据；缓存为空时触发刷新 |

### 请求报文

```http
GET /dashboard/member/overview HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer ${token}
```

### 出参说明

| 字段 | 类型 | 说明 |
|---|---|---|
| `code` | number | `200` 表示成功 |
| `msg` | string | 返回消息 |
| `data.stats` | object | 会员、会员卡、会员码展示统计 |
| `data.loginStats` | object | 小程序登录统计 |
| `data.visibleDeptIds` | array | 当前登录人可见部门 ID |
| `data.visibleDeptCount` | number | 当前登录人可见部门数量 |
| `data.missingDeptCount` | number | 可见部门中 Redis 暂无缓存的部门数量 |
| `data.scopeName` | string | 前端展示的数据范围名称 |
| `data.loginStatsScopeNote` | string | 登录统计口径提示 |
| `data.refreshedAt` | string | Redis 最后刷新时间 |
| `data.refreshStatus` | string | Redis 最后刷新状态 |

### `stats` 字段说明

| 字段 | 类型 | 说明 |
|---|---|---|
| `totalMembers` | number | 累计会员 |
| `borrowCardMembers` | number | 借阅卡会员 |
| `normalMembers` | number | 普通会员 |
| `unitBorrowCardMembers` | number | 单位借阅卡会员 |
| `naturalBorrowCardMembers` | number | 自然借阅卡会员 |
| `yearCardMembers` | number | 尊享会员 |
| `unitYearCardMembers` | number | 尊享会员中的单位会员 |
| `naturalYearCardMembers` | number | 尊享会员中的自然会员 |
| `halfYearCardMembers` | number | 畅享会员 |
| `unitHalfYearCardMembers` | number | 畅享会员中的单位会员 |
| `naturalHalfYearCardMembers` | number | 畅享会员中的自然会员 |
| `totalMemberCodeShowCount` | number | 会员码展示总量 |
| `yearMemberCodeShowCount` | number | 年会员码展示量 |
| `monthMemberCodeShowCount` | number | 月会员码展示量 |
| `yesterdayMemberCodeShowCount` | number | 昨日会员码展示量 |

### `loginStats` 字段说明

| 字段 | 类型 | 说明 |
|---|---|---|
| `totalLoginCount` | number | 用户登录总量 |
| `yearLoginCount` | number | 年用户登录量 |
| `monthLoginCount` | number | 月用户登录量 |
| `yesterdayLoginCount` | number | 昨日用户登录量 |

## 2. 手动刷新首页会员数据看板

### 基本信息

| 项目 | 内容 |
|---|---|
| 接口 | `POST /dashboard/member/refresh` |
| 权限 | admin 后台登录态 |
| 入参 | 无 |
| 数据范围 | 刷新全部部门预聚合缓存，查询时再按登录人权限合并 |
| 并发控制 | Redis 分布式锁 |

### 请求报文

```http
POST /dashboard/member/refresh HTTP/1.1
Host: 152.136.127.168
Authorization: Bearer ${token}
Content-Type: application/json

{}
```

### 成功响应

```json
{
  "code": 200,
  "msg": "刷新成功"
}
```

### 锁占用响应

```json
{
  "code": 601,
  "msg": "已有其他节点正在刷新，请稍后查看"
}
```

## 统计 SQL 口径

### 会员数据

- 主表：`member`
- 会员有效条件：`member.status = 0`
- 部门字段：`member.dept_id`
- 有效会员卡条件：`member_card.status = 1 and member_card.is_del = 0`
- 单位/自然会员拆分：`member.remark` 非空为单位会员，空为自然会员。
- 尊享/畅享拆分：当前按 `member_card.valid_days` 判断。

### 行为数据

- 日志表：`xhbs_api_log`
- 登录接口：`/api/mp/v1/auth/wechat-phone-login`
- 会员码接口：`/api/mp/v1/user/member-code`
- 成功条件：`response_code = 0`
- 年统计：`created_at >= 当年 1 月 1 日 00:00:00`
- 月统计：`created_at >= 当月 1 日 00:00:00`
- 昨日统计：`昨日 00:00:00 <= created_at < 今日 00:00:00`
