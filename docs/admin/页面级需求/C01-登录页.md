# C01-登录页

## 页面说明
管理后台登录页面，用户通过账号密码登录系统。

## 页面入口
- 浏览器直接访问管理后台地址（默认 http://localhost:80），未登录时自动跳转
- 登录过期时自动跳转

## 页面权限
无权限限制，所有用户可访问。

## 页面元素
- 系统Logo：左上角展示新华书店Logo
- 系统标题："新华书店会员借阅管理系统"
- 账号输入框
- 密码输入框（支持显示/隐藏切换）
- 验证码输入框 + 验证码图片（点击可刷新）
- 记住密码复选框
- 登录按钮
- 注册链接

## 页面交互

### 点击登录
1. 校验账号、密码、验证码是否为空
2. 调用 POST /login 接口
3. 成功后获取 Token，存储到 localStorage
4. 调用 GET /getInfo 获取用户信息和权限
5. 调用 GET /getRouters 获取菜单路由
6. 前端动态添加路由，跳转首页仪表盘

### 点击验证码图片 → 刷新验证码
### 回车键 → 等同于点击登录
### 点击注册链接 → 跳转注册页

## 字段说明
| 字段名称 | 类型 | 是否必填 | 说明 |
|----------|------|----------|------|
| username | string | 是 | 登录账号 |
| password | string | 是 | 登录密码 |
| code | string | 是 | 图片验证码 |
| uuid | string | 是 | 验证码唯一标识 |
| rememberMe | boolean | 否 | 是否记住密码 |

## 接口
- GET /captchaImage — 获取验证码
- POST /login — 登录
- GET /getInfo — 获取用户信息
- GET /getRouters — 获取路由菜单

## 数据校验
| 规则 | 说明 |
|------|------|
| 账号必填 | 非空校验 |
| 密码必填 | 非空校验 |
| 验证码必填 | 非空校验 |

## 异常处理
| 场景 | 处理方式 |
|------|----------|
| 账号或密码错误 | 提示"用户名或密码错误"，刷新验证码 |
| 验证码错误/过期 | 提示错误，刷新验证码 |
| 账号被停用 | 提示"账号已被停用" |
| 网络异常 | 提示"网络连接失败" |

## UI状态
- Loading：登录按钮显示加载中，禁止重复提交
- Error：输入框下方红色提示
- Success：跳转首页
