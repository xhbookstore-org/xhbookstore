# S04-部门管理

## 页面说明
管理组织机构（公司、部门、小组），同时作为门店数据使用（sys_dept表）。

## 页面入口
- 侧边栏菜单 → 系统管理 → 部门管理

## 页面权限
- system:dept:list / system:dept:add / system:dept:edit / system:dept:remove

## 页面元素
- 左侧部门树
- 右侧部门详情
- 操作按钮：新增、编辑、删除

## 页面交互
### 部门树 → 点击查看详情
### 新增 → 选上级部门，填名称、负责人、电话、邮箱、排序
### 编辑 → 修改部门信息
### 删除 → 二次确认，含子部门/已关联用户不可删除

## 字段说明
| 字段名称 | 类型 | 是否必填 | 说明 |
|----------|------|----------|------|
| parentId | long | 是 | 上级部门ID |
| deptName | string | 是 | 部门/门店名称 |
| leader | string | 否 | 负责人 |
| phone | string | 否 | 电话 |
| email | string | 否 | 邮箱 |
| orderNum | int | 是 | 排序 |
| status | string | 是 | 状态 |

## 接口
- GET /system/dept/list / GET /system/dept/{id} / POST /system/dept / PUT /system/dept / DELETE /system/dept/{id}

## 特别说明
部门数据同时作为门店数据使用。会员开卡门店、员工所属门店均关联此表。
