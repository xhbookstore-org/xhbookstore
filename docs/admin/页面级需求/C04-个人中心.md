# C04-个人中心

## 页面说明
当前登录用户查看和修改个人信息。

## 页面入口
- 顶部导航栏右侧用户头像下拉 → "个人中心"

## 页面权限
所有已登录用户可访问。

## 页面元素
- 用户头像（支持修改）
- 基本信息区（用户名、手机号、邮箱、创建时间等）
- 修改密码区

## 页面交互
### 修改头像 → 弹出文件选择器 → 裁剪 → 上传
### 修改基本信息 → 编辑 → 保存 → 刷新
### 修改密码 → 旧密码+新密码+确认 → 成功退出重新登录

## 字段说明
| 字段名称 | 类型 | 是否必填 | 说明 |
|----------|------|----------|------|
| nickName | string | 否 | 昵称 |
| phonenumber | string | 否 | 手机号 |
| email | string | 否 | 邮箱 |
| sex | string | 否 | 性别 |
| oldPassword | string | 是(改密码) | 旧密码 |
| newPassword | string | 是(改密码) | 新密码 |

## 接口
- GET /system/user/profile — 获取个人信息
- PUT /system/user/profile — 修改个人信息
- PUT /system/user/profile/updatePwd — 修改密码
- POST /system/user/profile/avatar — 上传头像

## 异常处理
| 场景 | 处理方式 |
|------|----------|
| 旧密码错误 | 提示"旧密码输入错误" |
| 头像上传失败 | 提示重试 |
