# S02-角色管理

## 页面说明
管理系统角色，配置角色菜单权限和数据权限范围。

## 页面入口
- 侧边栏菜单 → 系统管理 → 角色管理

## 页面权限
- system:role:list / system:role:add / system:role:edit / system:role:remove

## 页面元素
- 查询条件：角色名称、权限字符、状态、创建时间
- 数据列表：序号、角色名称、权限字符、显示顺序、状态、创建时间、操作（编辑/删除/分配用户）
- 分页组件

## 页面交互
### 新增/编辑 → 弹窗：角色名称、权限字符、排序、状态 + 菜单权限树 + 数据范围选择
### 删除 → 二次确认，已关联用户不可删除
### 分配用户 → 弹窗：左侧未分配/右侧已分配，支持批量操作

## 字段说明
| 字段名称 | 类型 | 是否必填 | 说明 |
|----------|------|----------|------|
| roleName | string | 是 | 角色名称 |
| roleKey | string | 是 | 权限字符（唯一） |
| roleSort | int | 是 | 显示顺序 |
| status | string | 是 | 状态 |
| menuIds | long[] | 是 | 关联菜单ID |
| dataScope | string | 是 | 数据范围(1全部/2自定义/3本部门/4本部门及以下/5仅本人) |

## 接口
- GET /system/role/list / GET /system/role/{id} / POST /system/role / PUT /system/role / DELETE /system/role/{ids}
