# S06-字典管理

## 页面说明
管理系统中的字典类型和字典数据，统一维护下拉选项等固定数据。

## 页面入口
- 侧边栏菜单 → 系统管理 → 字典管理

## 页面权限
- system:dict:list / system:dict:add / system:dict:edit / system:dict:remove

## 页面元素
- 查询条件：字典名称、字典类型、状态
- 数据列表：序号、字典名称、字典类型、状态、备注、创建时间、操作（编辑/删除/字典数据）
- 分页组件

## 页面交互
### 字典类型 → 新增/编辑/删除
### 字典数据 → 进入子页面管理键值对
### 刷新缓存 → 重载Redis缓存

## 字段说明（字典类型）
| 字段名称 | 类型 | 是否必填 | 说明 |
|----------|------|----------|------|
| dictName | string | 是 | 字典名称 |
| dictType | string | 是 | 字典类型标识 |
| status | string | 是 | 状态 |

## 字段说明（字典数据）
| 字段名称 | 类型 | 是否必填 | 说明 |
|----------|------|----------|------|
| dictLabel | string | 是 | 字典标签 |
| dictValue | string | 是 | 字典值 |
| dictSort | int | 是 | 排序 |
| isDefault | string | 是 | 是否默认 |

## 接口
字典类型：GET/POST/PUT/DELETE /system/dict/type/...
字典数据：GET/POST/PUT/DELETE /system/dict/data/...
