# S08-通知公告

## 页面说明
管理系统通知公告的发布和维护。

## 页面入口
- 侧边栏菜单 → 系统管理 → 通知公告

## 页面权限
- system:notice:list / system:notice:add / system:notice:edit / system:notice:remove

## 页面元素
- 查询条件：公告标题、操作人员、公告类型
- 数据列表：序号、公告标题、公告类型、状态、创建者、创建时间、操作（编辑/删除/查看已读用户）
- 分页组件

## 页面交互
### 新增/编辑 → 弹窗填写标题、类型、状态、内容（富文本编辑器）
### 查看已读用户 → 弹窗展示已读用户列表
### 删除 → 二次确认

## 字段说明
| 字段名称 | 类型 | 是否必填 | 说明 |
|----------|------|----------|------|
| noticeTitle | string | 是 | 公告标题 |
| noticeType | string | 是 | 公告类型 |
| noticeContent | string | 是 | 公告内容 |
| status | string | 是 | 状态 |

## 接口
- GET/POST/PUT/DELETE /system/notice/...

## 待确认问题
