# T05-缓存监控

## 页面说明
Redis缓存信息查询和命令统计。

## 页面入口
- 侧边栏菜单 → 系统监控 → 缓存监控

## 页面权限
- monitor:cache:list

## 页面元素
- 缓存信息区：Redis基本信息（版本、运行天数、内存、客户端数等）
- 命令统计区：Redis命令使用统计
- 缓存列表（子页面）：键名列表、模糊搜索、删除缓存

## 页面交互
### 查看缓存信息 → 自动加载Redis实时数据
### 缓存列表 → 切换Tab查看键名、搜索、删除

## 接口
- GET /monitor/cache / GET /monitor/cache/getNames / GET /monitor/cache/getKeys/{cacheName} / DELETE /monitor/cache/clearCacheName/{cacheName}
