# M03 会员卡订单

## 页面目标

展示会员卡售卡订单，用于收入查询和对账。

## 查询条件

| 条件 | 类型 | 说明 |
|---|---|---|
| 订单号 | 输入框 | 售卡订单号 |
| 会员姓名 | 输入框 | 模糊查询 |
| 会员编号 | 输入框 | 会员编号 |
| 门店 | 下拉框 | 售卡门店 |
| 订单状态 | 下拉框 | 已支付、已退款 |
| 付款时间 | 日期范围 | 按 pay_time 查询 |

## 表格字段

| 字段 | 说明 |
|---|---|
| ID | 订单主键 |
| 售卡订单号 | 唯一订单号 |
| 会员姓名 | 会员姓名 |
| 会员编号 | 会员编号 |
| 卡类型 | 卡类型快照 |
| 应收金额 | 原金额 |
| 实收金额 | 员工录入金额 |
| 支付方式 | 当前为员工录入 |
| 订单状态 | 已支付、已退款 |
| 支付时间 | 付款时间 |
| 门店 | 售卡门店 |
| 开单员工 | 操作人 |

## 对账要求

- 售卡订单必须能关联到会员卡记录。
- 已退款订单必须能关联到退卡订单。
- 订单金额与会员卡 `sale_amount` 保持一致。
