# C03-首页仪表盘

## 页面说明
登录后的默认首页，展示系统概览数据和图表统计。

## 页面入口
- 登录成功后自动跳转
- 侧边栏菜单"首页"

## 页面权限
所有已登录用户可访问。

## 页面元素
- 顶部统计面板（关键数据指标卡片）
- 柱状图、折线图、饼图、雷达图

## 页面交互
### 加载页面 → 请求首页统计数据 → 渲染图表
### 图表hover提示、自适应窗口大小

## 接口
- GET /index — 获取首页统计数据（当前为Mock数据）

## 异常处理
| 场景 | 处理方式 |
|------|----------|
| 数据加载失败 | 保留上次数据，Toast提示 |
