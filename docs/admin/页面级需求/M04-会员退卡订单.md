# M04 会员退卡订单

## 页面目标

展示会员卡退卡订单，用于退款查询和对账。

## 查询条件

| 条件 | 类型 | 说明 |
|---|---|---|
| 退卡订单号 | 输入框 | 退卡订单唯一号 |
| 原售卡订单号 | 输入框 | 关联售卡订单 |
| 会员姓名 | 输入框 | 模糊查询 |
| 会员编号 | 输入框 | 会员编号 |
| 门店 | 下拉框 | 退卡操作门店 |
| 退款时间 | 日期范围 | 按 refund_time 查询 |

## 表格字段

| 字段 | 说明 |
|---|---|
| ID | 退卡订单主键 |
| 退卡订单号 | 唯一退款单号 |
| 原售卡订单号 | 原订单 |
| 会员姓名 | 会员姓名 |
| 会员编号 | 会员编号 |
| 卡类型 | 卡类型快照 |
| 退款金额 | 实退金额 |
| 退款方式 | CASH 等 |
| 退款时间 | 操作时间 |
| 退款原因 | 员工填写 |
| 门店 | 操作门店 |
| 操作人 | 退卡员工 |

## 对账要求

- 每条退卡订单必须关联一条售卡订单。
- 每条退卡订单必须关联一张会员卡。
- 会员卡状态必须为已退款。
- 原售卡订单状态必须为已退款。
