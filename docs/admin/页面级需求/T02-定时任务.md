# T02-定时任务

## 页面说明
在线管理定时任务（新增/修改/删除/暂停/执行），查看调度日志。

## 页面入口
- 侧边栏菜单 → 系统监控 → 定时任务

## 页面权限
- monitor:job:list / monitor:job:add / monitor:job:edit / monitor:job:remove / monitor:job:changeStatus / monitor:job:run

## 页面元素
- 查询条件：任务名称、任务组名、状态
- 数据列表：序号、任务名称、任务组名、调用目标、cron表达式、状态、操作（编辑/删除/暂停-恢复/执行一次/调度日志）
- 分页组件

## 页面交互
### 新增/编辑 → 弹窗：任务名称、任务组名、调用目标(Bean方法)、cron表达式、并发策略、状态
### 暂停/恢复 → 切换任务状态
### 执行一次 → 立即触发
### 调度日志 → 子页面展示执行历史和异常信息

## 字段说明
| 字段名称 | 类型 | 是否必填 | 说明 |
|----------|------|----------|------|
| jobName | string | 是 | 任务名称 |
| jobGroup | string | 是 | 任务组名 |
| invokeTarget | string | 是 | 调用目标(Bean方法) |
| cronExpression | string | 是 | Cron表达式 |
| concurrent | string | 是 | 是否并发执行 |
| status | string | 是 | 状态 |

## 接口
- GET /monitor/job/list / GET /monitor/job/{id} / POST /monitor/job / PUT /monitor/job / DELETE /monitor/job/{ids} / PUT /monitor/job/changeStatus / PUT /monitor/job/run / GET /monitor/job/log/list
