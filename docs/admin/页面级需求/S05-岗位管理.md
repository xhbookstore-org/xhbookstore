# S05-岗位管理

## 页面说明
配置系统用户所属职务/岗位。

## 页面入口
- 侧边栏菜单 → 系统管理 → 岗位管理

## 页面权限
- system:post:list / system:post:add / system:post:edit / system:post:remove

## 页面元素
- 查询条件：岗位编码、岗位名称、状态
- 数据列表：序号、岗位编码、岗位名称、显示顺序、状态、创建时间、操作（编辑/删除）
- 分页组件

## 页面交互
### 新增/编辑 → 弹窗填写岗位编码、名称、排序、状态
### 删除 → 二次确认，已关联用户不可删除

## 字段说明
| 字段名称 | 类型 | 是否必填 | 说明 |
|----------|------|----------|------|
| postCode | string | 是 | 岗位编码 |
| postName | string | 是 | 岗位名称 |
| postSort | int | 是 | 排序 |
| status | string | 是 | 状态 |

## 接口
- GET /system/post/list / GET /system/post/{id} / POST /system/post / PUT /system/post / DELETE /system/post/{ids}
