# S01-用户管理

## 页面说明
管理后台系统用户（员工账号）的增删改查、角色分配。

## 页面入口
- 侧边栏菜单 → 系统管理 → 用户管理

## 页面权限
- 权限标识：system:user:list / system:user:add / system:user:edit / system:user:remove / system:user:export

## 页面元素
- 查询条件：用户名、手机号、状态、部门（树形选择）、创建时间范围
- 操作按钮：新增、导出
- 数据列表：序号、用户名称、登录名称、部门、手机号、状态、创建时间、操作（编辑/删除/重置密码/分配角色）
- 分页组件

## 页面交互
### 查询 → 多条件组合筛选
### 新增用户 → 弹窗填写：用户名、密码、姓名、部门、手机号、邮箱、性别、角色、状态
### 编辑用户 → 弹窗编辑（不允许修改登录名）
### 删除用户 → 二次确认，不允许删除自己
### 重置密码 → 二次确认，重置为初始密码
### 分配角色 → 弹窗勾选角色
### 导出 → 当前筛选条件全量导出

## 字段说明
| 字段名称 | 类型 | 是否必填 | 说明 |
|----------|------|----------|------|
| userName | string | 是 | 登录账号 |
| nickName | string | 是 | 用户名称 |
| password | string | 是(新增) | 密码 |
| deptId | long | 是 | 所属部门 |
| phonenumber | string | 否 | 手机号 |
| email | string | 否 | 邮箱 |
| sex | string | 否 | 性别 |
| status | string | 是 | 状态（0正常，1停用） |
| roleIds | long[] | 否 | 角色ID列表 |

## 接口
- GET /system/user/list / GET /system/user/{id} / POST /system/user / PUT /system/user / DELETE /system/user/{ids} / PUT /system/user/resetPwd / GET /system/user/export

## 数据校验
用户名必填+唯一、密码必填(新增)、手机号/邮箱格式校验

## 异常处理
用户名已存在提示、不允许删除自己、空状态

## 待确认问题
无
