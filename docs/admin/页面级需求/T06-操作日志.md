# T06-操作日志

## 页面说明
系统正常操作日志记录和查询。

## 页面入口
- 侧边栏菜单 → 系统监控 → 操作日志

## 页面权限
- monitor:operlog:list / monitor:operlog:remove

## 页面元素
- 查询条件：系统模块、操作人员、操作类型、操作状态、操作时间
- 操作按钮：删除、清空、导出
- 数据列表：序号、系统模块、操作类型、请求方式、操作人员、操作地址、操作状态、操作时间、操作（详情）
- 分页组件

## 页面交互
### 查看详情 → 弹窗：操作模块、请求URL、操作方法、请求参数、返回参数、异常信息、耗时
### 删除 → 支持批量删除
### 清空 → 二次确认

## 字段说明
| 字段名称 | 类型 | 说明 |
|----------|------|------|
| title | string | 系统模块 |
| operName | string | 操作人员 |
| businessType | int | 操作类型 |
| requestMethod | string | 请求方式 |
| operIp | string | 操作IP |
| status | int | 状态(0成功1失败) |
| operTime | date | 操作时间 |
| costTime | long | 耗时(ms) |

## 接口
- GET /monitor/operlog/list / DELETE /monitor/operlog/{ids} / DELETE /monitor/operlog/clean / GET /monitor/operlog/export
