# T01-在线用户

## 页面说明
监控当前系统中活跃用户状态，支持强退操作。

## 页面入口
- 侧边栏菜单 → 系统监控 → 在线用户

## 页面权限
- monitor:online:list / monitor:online:forceLogout

## 页面元素
- 查询条件：用户名、登录地址
- 数据列表：序号、会话编号、用户名、部门、登录地址、登录地点、浏览器、操作系统、登录时间、操作（强退）
- 分页组件

## 页面交互
### 强退 → 二次确认后强制清除服务端Session

## 接口
- GET /monitor/online/list / DELETE /monitor/online/{tokenId}

## 待确认问题
无
