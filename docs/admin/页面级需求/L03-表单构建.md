# L03-表单构建

## 页面说明
可视化拖拽表单设计器，拖动表单元素生成Vue代码。

## 页面入口
- 侧边栏菜单 → 系统工具 → 表单构建

## 页面权限
- tool:build:list

## 页面元素
- 左侧组件面板（拖拽源）
- 中间画布区（实时预览）
- 右侧属性面板（编辑选中组件属性）

## 页面交互
### 拖拽组件 → 从左侧拖到画布
### 编辑属性 → 选中组件编辑标签名、字段名、默认值、校验规则、宽度等
### 生成代码 → 导出Vue文件代码
### 清空/回退 → 清空画布、撤销

## 接口
- 纯前端功能
