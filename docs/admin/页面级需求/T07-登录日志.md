# T07-登录日志

## 页面说明
系统登录日志记录查询，包含登录异常记录。

## 页面入口
- 侧边栏菜单 → 系统监控 → 登录日志

## 页面权限
- monitor:logininfor:list / monitor:logininfor:remove

## 页面元素
- 查询条件：用户名、登录状态、登录时间
- 操作按钮：删除、清空、导出
- 数据列表：序号、用户名、登录地址、登录地点、浏览器、操作系统、登录状态、操作信息、登录时间
- 分页组件

## 页面交互
### 查询/删除/清空（同操作日志）

## 字段说明
| 字段名称 | 类型 | 说明 |
|----------|------|------|
| userName | string | 用户名 |
| ipaddr | string | 登录IP |
| loginLocation | string | 登录地点 |
| browser | string | 浏览器 |
| os | string | 操作系统 |
| status | int | 状态(0成功1失败) |
| msg | string | 操作信息 |
| loginTime | date | 登录时间 |

## 接口
- GET /monitor/logininfor/list / DELETE /monitor/logininfor/{ids} / DELETE /monitor/logininfor/clean / GET /monitor/logininfor/export
