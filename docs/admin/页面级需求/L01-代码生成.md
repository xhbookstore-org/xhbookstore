# L01-代码生成

## 页面说明
根据数据库表结构自动生成前后端代码（Java/Vue/SQL/Mapper XML），支持CRUD下载。

## 页面入口
- 侧边栏菜单 → 系统工具 → 代码生成

## 页面权限
- tool:gen:list / tool:gen:edit

## 页面元素
- 查询条件：表名称、表描述
- 操作按钮：导入、创建表
- 数据列表：表名称、表描述、实体类名、创建时间、更新时间、操作（预览/编辑/删除/生成代码）
- 分页组件

## 页面交互
### 导入表 → 弹窗展示数据库所有表，勾选导入
### 编辑配置 → 弹窗：表名、实体名、作者、模板、包路径、模块名、业务名、功能名、上级菜单、字段配置
### 预览代码 → 弹窗Tab展示生成代码
### 生成代码 → 下载zip包
### 创建表 → 在线可视化建表

## 字段说明
| 字段名称 | 类型 | 是否必填 | 说明 |
|----------|------|----------|------|
| tableName | string | 是 | 数据库表名 |
| tableComment | string | 是 | 表描述 |
| className | string | 是 | 实体类名 |
| functionAuthor | string | 是 | 作者 |
| tplCategory | string | 是 | 模板类型(crud/tree) |
| packageName | string | 是 | 生成包路径 |
| moduleName | string | 是 | 生成模块名 |
| businessName | string | 是 | 生成业务名 |

## 接口
- GET /tool/gen/list / GET /tool/gen/{id} / PUT /tool/gen / DELETE /tool/gen/{ids} / POST /tool/gen/importTable / GET /tool/gen/preview/{id} / GET /tool/gen/genCode/{tableName}
