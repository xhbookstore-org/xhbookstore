# S03-菜单管理

## 页面说明
管理系统侧边栏菜单结构，支持目录(M)、菜单(C)、按钮(F)三种类型。

## 页面入口
- 侧边栏菜单 → 系统管理 → 菜单管理

## 页面权限
- system:menu:list / system:menu:add / system:menu:edit / system:menu:remove

## 页面元素
- 左侧菜单树（全部菜单树形展示）
- 右侧菜单详情
- 操作按钮：新增、编辑、删除、展开/折叠全部

## 页面交互
### 菜单树 → 点击节点展开/收起，查看详情
### 新增菜单 → 选择类型(M/C/F)，选择上级菜单，填写路由/组件/权限标识等
### 编辑菜单 → 修改菜单信息即时生效
### 删除 → 二次确认，含子菜单不可删除

## 字段说明
| 字段名称 | 类型 | 是否必填 | 说明 |
|----------|------|----------|------|
| parentId | long | 是 | 上级菜单ID |
| menuName | string | 是 | 菜单名称 |
| menuType | string | 是 | M目录/C菜单/F按钮 |
| orderNum | int | 是 | 显示顺序 |
| path | string | C类型必填 | 路由地址 |
| component | string | 否 | 组件路径 |
| perms | string | F类型必填 | 权限标识 |
| icon | string | 否 | 图标 |
| visible | string | 是 | 是否显示 |
| status | string | 是 | 状态 |

## 接口
- GET /system/menu/list / GET /system/menu/{id} / POST /system/menu / PUT /system/menu / DELETE /system/menu/{id}
