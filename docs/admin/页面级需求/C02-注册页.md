# C02-注册页

## 页面说明
用户自助注册管理后台账号页面。

## 页面入口
- 登录页底部"注册"链接

## 页面权限
无权限限制（取决于系统配置是否开放注册）。

## 页面元素
- 注册表单：账号、密码、确认密码
- 返回登录链接

## 页面交互
### 提交注册 → 校验 → 调用 POST /register → 成功跳转登录页
### 点击返回登录 → 跳转登录页

## 字段说明
| 字段名称 | 类型 | 是否必填 | 说明 |
|----------|------|----------|------|
| username | string | 是 | 登录账号 |
| password | string | 是 | 密码（最短6位） |
| confirmPassword | string | 是 | 确认密码 |

## 接口
- POST /register — 用户注册

## 数据校验
| 规则 | 说明 |
|------|------|
| 账号必填 + 唯一 | 全局查重 |
| 密码必填 + 长度≥6 | 必填校验 |
| 确认密码 | 必须与密码一致 |

## 异常处理
| 场景 | 处理方式 |
|------|----------|
| 账号已存在 | 提示"账号已被注册" |
| 密码不一致 | 提示"两次输入的密码不一致" |
| 注册功能关闭 | 提示"系统暂未开放注册" |

## 待确认问题
- 是否需要手机号/邮箱验证？是否需要管理员审核？
