# S07-参数配置

## 页面说明
管理系统动态配置参数，键值对形式。

## 页面入口
- 侧边栏菜单 → 系统管理 → 参数配置

## 页面权限
- system:config:list / system:config:add / system:config:edit / system:config:remove

## 页面元素
- 查询条件：参数名称、参数键名、系统内置
- 数据列表：序号、参数名称、参数键名、参数键值、系统内置、备注、创建时间、操作（编辑/删除）
- 分页组件

## 页面交互
### 新增/编辑 → 弹窗填写参数名称、键名、键值、是否内置、备注
### 删除 → 系统内置参数不可删除
### 刷新缓存 → 重载参数到Redis

## 字段说明
| 字段名称 | 类型 | 是否必填 | 说明 |
|----------|------|----------|------|
| configName | string | 是 | 参数名称 |
| configKey | string | 是 | 参数键名 |
| configValue | string | 是 | 参数键值 |
| configType | string | 是 | 系统内置(Y/N) |

## 接口
- GET/POST/PUT/DELETE /system/config/...
