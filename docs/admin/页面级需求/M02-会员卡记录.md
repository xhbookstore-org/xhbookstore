# M02 会员卡记录

## 页面目标

展示会员卡队列全量记录，支持按会员、卡状态、门店、付款时间查询，便于维护生效卡、待生效卡、已过期卡和已退款卡。

## 查询条件

| 条件 | 类型 | 说明 |
|---|---|---|
| 会员姓名 | 输入框 | 模糊查询 |
| 会员编号 | 输入框 | 精确或模糊查询 |
| 卡状态 | 下拉框 | 待生效、生效中、已过期、已退款 |
| 门店 | 下拉框 | 按开卡门店过滤 |
| 付款时间 | 日期范围 | 按 paid_at 查询 |

## 表格字段

| 字段 | 说明 |
|---|---|
| ID | 会员卡主键 |
| 会员姓名 | 会员名称 |
| 会员编号 | 会员卡号/会员编号 |
| 卡类型 | 卡类型快照 |
| 状态 | 待生效、生效中、已过期、已退款 |
| 售卡订单号 | 来源订单 |
| 付款时间 | 7 天可退判断依据 |
| 生效时间 | 卡实际开始时间 |
| 到期时间 | 卡实际结束时间 |
| 实收金额 | 售卡实收 |
| 退卡订单号 | 已退款时展示 |

## 操作

| 操作 | 权限 | 说明 |
|---|---|---|
| 导出 | `member:card:export` | 按当前查询条件导出全量会员卡记录，文件用于外部 ERP 录入和对账 |
| 日志 | `member:card:query` | 查看会员卡状态变化日志 |
| 退卡 | `member:card:refund` | 仅付款 7 天内且未退款的卡展示 |

## 导出规则

- 导出文件名：`会员卡记录_查询条件_yyyyMMddHHmmss.xlsx`。
- 导出范围：符合当前查询条件的全量数据，不受分页影响。
- 导出字段：会员卡记录 ID、会员编号、会员姓名、手机号、卡类型、有效天数、卡状态、是否当前生效、售卡订单号、实收金额、付款时间、生效时间、到期时间、退款订单号、退款时间、门店、操作员工、备注、创建时间、更新时间。
- 待生效卡：状态展示为“待生效”，“是否当前生效”为“否”，生效/到期时间展示会员卡队列顺延后的预计时间。
- 已退款卡：状态展示为“已退款”，携带退款订单号和退款时间。

## 默认排序

优先展示生效中、待生效、已退款、已过期，再按付款/生效/创建时间倒序和 ID 倒序排列，保证最近业务记录优先可见。
