# M02 会员卡记录

## 页面目标

展示会员卡队列全量记录，支持按会员、卡状态、门店、付款时间查询，便于维护生效卡、待生效卡、已过期卡和已退款卡。

## 查询条件

| 条件 | 类型 | 说明 |
|---|---|---|
| 会员姓名 | 输入框 | 模糊查询 |
| 会员编号 | 输入框 | 精确或模糊查询 |
| 卡状态 | 下拉框 | 待生效、生效中、已过期、已退款 |
| 门店 | 下拉框 | 按开卡门店过滤 |
| 付款时间 | 日期范围 | 按 paid_at 查询 |

## 表格字段

| 字段 | 说明 |
|---|---|
| ID | 会员卡主键 |
| 会员姓名 | 会员名称 |
| 会员编号 | 会员卡号/会员编号 |
| 卡类型 | 卡类型快照 |
| 状态 | 待生效、生效中、已过期、已退款 |
| 售卡订单号 | 来源订单 |
| 付款时间 | 7 天可退判断依据 |
| 生效时间 | 卡实际开始时间 |
| 到期时间 | 卡实际结束时间 |
| 实收金额 | 售卡实收 |
| 退卡订单号 | 已退款时展示 |

## 操作

| 操作 | 权限 | 说明 |
|---|---|---|
| 日志 | `member:card:query` | 查看会员卡状态变化日志 |
| 退卡 | `member:card:refund` | 仅付款 7 天内且未退款的卡展示 |

## 默认排序

优先按会员、付款时间、ID 排序，保证同一会员的卡队列顺序可读。
