# 会员扩展表 (member_ext)

## 说明
会员扩展信息表，与member表1:1关联，存储会员的补充信息。

## 字段定义
| 字段 | 类型 | 约束 | 是否必填 | 说明 | 来源 |
|------|------|------|----------|------|------|
| id | int PK | — | 自动 | 主键 | 系统生成 |
| member_id | int | UNIQUE FK→member CASCADE | 是 | 关联会员ID | 系统关联 |
| gender | enum('男','女') | — | 否 | 性别 | 手动录入/Excel导入 |
| age | int | — | 否 | 年龄 | 手动录入/Excel导入 |
| unit_phone | varchar(20) | — | 否 | 单位电话 | 手动录入/Excel导入 |
| wechat | varchar(50) | — | 否 | 微信号 | 微信绑定 |
| weibo | varchar(50) | — | 否 | 微博账号 | 手动录入 |
| join_date | date | — | 是 | 入会日期 | 系统自动（当天） |
| total_points | int | DEFAULT 0 | 是 | 累计总积分 | 积分系统维护 |
| level_points | int | DEFAULT 0 | 是 | 级别积分 | 积分系统维护 |
| discount | decimal(5,2) | — | 否 | 当前折扣 | 会员级别关联 |
| total_purchase_amount | decimal(10,2) | DEFAULT 0 | 是 | 累计购书总额 | 购书系统维护 |
| total_purchase_count | int | DEFAULT 0 | 是 | 累计购书册数 | 购书系统维护 |
| total_purchase_times | int | DEFAULT 0 | 是 | 累计购书次数 | 购书系统维护 |
| superior_name | varchar(50) | — | 否 | 推荐人姓名 | 手动录入 |
| superior_points_ratio | decimal(5,2) | — | 否 | 上级积分比例 | 系统设置 |
| business_staff_name | varchar(50) | — | 是 | 业务员姓名（开卡员工） | 系统自动记录 |
| excel_raw_data | longtext | — | 否 | 导入Excel原始JSON快照 | Excel导入时自动 |
| created_at | datetime | — | 是 | 创建时间 | 系统自动 |
| updated_at | datetime | — | 是 | 更新时间 | 系统自动 |

## 关联关系
- member_id → member.id（1:1，CASCADE删除）
