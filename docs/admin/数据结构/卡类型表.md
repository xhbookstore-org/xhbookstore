# 卡类型表 (card_type)

## 说明
定义会员卡的类型，每种类型对应不同的有效期天数，支持区分新卡和续费类型。

## 字段定义
| 字段 | 类型 | 约束 | 是否必填 | 说明 | 来源 |
|------|------|------|----------|------|------|
| id | int PK | — | 自动 | 主键 | 系统生成 |
| type_name | varchar(50) | — | 是 | 类型名称 | 系统初始化 |
| valid_days | int | — | 是 | 有效天数 | 系统初始化 |
| is_renewal | tinyint | — | 是 | 是否续费类型（0-新卡，1-续费） | 系统初始化 |
| status | tinyint | — | 是 | 状态（0-正常，1-停用） | 系统管理 |

## 初始数据
| type_name | valid_days | is_renewal | 说明 |
|-----------|------------|------------|------|
| 年卡 | 365 | 0 | 新卡，有效期365天 |
| 半年卡 | 180 | 0 | 新卡，有效期180天 |
| 续费年卡 | 365 | 1 | 续费，在原有效期上叠加365天 |
| 续费半年卡 | 180 | 1 | 续费，在原有效期上叠加180天 |

## 业务规则
- is_renewal=0：新卡类型，有效期从当前日期开始计算
- is_renewal=1：续费类型，有效期在原valid_date基础上叠加

## 关联关系
- id → member.card_type_id
