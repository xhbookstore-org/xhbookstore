# 图书相关表

> 状态：表结构已设计，Service/Controller待建

## 图书主表 (book_info)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint PK | 主键 |
| book_name | varchar(200) | 图书名称 |
| isbn | varchar(30) | ISBN编号 |
| author | varchar(100) | 作者 |
| publisher | varchar(200) | 出版社 |
| price | decimal(10,2) | 定价 |
| sale_price | decimal(10,2) | 售价 |
| stock_qty | int | 库存数量 |
| lendable_qty | int | 可借数量 |
| book_status | tinyint | 1-初始化 2-上架 3-下架 |
| cover_url | varchar(500) | 封面图URL |
| is_del | tinyint | 删除标识 |

## 借书单主表 (book_borrow_order)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint PK | 主键 |
| order_no | varchar(50) UNIQUE | 借书单号(DY开头) |
| member_id | int FK | 会员ID |
| total_book_count | int | 借书总数 |
| borrow_status | tinyint | 1借阅中/2已全还/3部分还/4部分转购/5全转购 |
| borrow_time | datetime | 借书时间 |
| expected_return_time | datetime | 预计还书时间 |
| is_del | tinyint | 删除标识 |

## 借书明细表 (book_borrow_detail)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint PK | 主键 |
| borrow_order_id | bigint FK | 关联借书单 |
| book_id | bigint | 图书ID |
| book_name | varchar(200) | 图书名称 |
| borrow_qty | int | 借书数量 |
| returned_qty | int | 已还数量 |
| purchase_qty | int | 已转购数量 |
| borrow_status | tinyint | 借阅状态 |

## 还书明细表 (book_return_detail)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint PK | 主键 |
| return_order_no | varchar(50) | 还书单号(HS开头) |
| borrow_detail_id | bigint FK | 关联借书明细 |
| return_qty | int | 还书数量 |
| return_type | tinyint | 1-正常 2-损坏 3-遗失 |
| return_time | datetime | 还书时间 |

## 借转购订单表 (dd_book_purchase_order)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint PK | 主键 |
| order_no | varchar(50) | 购书单号(JSDD开头) |
| order_type | tinyint | 1-直接购书 2-借转购 |
| borrow_detail_id | bigint | 关联借书明细 |
| qty | int | 数量 |
| receivable_amount | decimal(10,2) | 应收金额 |
| order_status | tinyint | 1-已付款 2-已取消 3-已退款 |

## 订单号规则
| 前缀 | 表 | 示例 |
|------|-----|------|
| DY | book_borrow_order | DY20260630143000123456 |
| HS | book_return_detail | HS20260630143000654321 |
| JSDD | dd_book_purchase_order | JSDD20260630143000789012 |
| IN | xhbs_points_order(入账) | IN20260628120000123456 |
| OT | xhbs_points_order(出账) | OT20260628120000654321 |

## 数量闭环校验
核心公式：borrow_qty = returned_qty + purchase_qty + 未处理
三层验证：派生字段vs来源表 / 状态vs数量 / 边界约束
