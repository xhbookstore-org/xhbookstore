# 积分相关表

## 积分订单表 (xhbs_points_order)
积分变动的主记录表。

| 字段 | 类型 | 约束 | 是否必填 | 说明 | 来源 |
|------|------|------|----------|------|------|
| id | int PK | — | 自动 | 主键 | 系统生成 |
| order_no | varchar(50) | UNIQUE | 是 | 订单号(IN/OT开头) | 系统生成 |
| member_id | int | FK→member | 是 | 会员ID | 关联 |
| direction | varchar(10) | — | 是 | add/deduct | 操作决定 |
| points | int | — | 是 | 变动积分数 | 手动输入 |
| before_points | int | — | 是 | 操作前积分 | 系统快照 |
| after_points | int | — | 是 | 操作后积分 | 系统计算 |
| reason | varchar(100) | — | 是 | 变动事由 | 积分事项选择 |
| device | varchar(20) | — | 是 | PC/小程序/POS | 系统标记 |
| operator | varchar(50) | — | 是 | 操作人 | 系统记录 |
| remark | varchar(255) | — | 否 | 备注 | 手动输入 |
| created_at | datetime | — | 是 | 创建时间 | 系统自动 |

### 订单号规则
| 方向 | 前缀 | 示例 |
|------|------|------|
| add(增加) | IN | IN20260628120000123456 |
| deduct(消耗) | OT | OT20260628120000654321 |

## 积分入账单明细表 (xhbs_points_user_into_bill_detail)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | int PK | 主键 |
| order_id | int FK | 关联订单ID |
| member_id | int FK | 会员ID |
| points | int | 入账积分 |
| created_at | datetime | 创建时间 |

## 积分出账单明细表 (xhbs_points_user_out_bill_detail)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | int PK | 主键 |
| order_id | int FK | 关联订单ID |
| member_id | int FK | 会员ID |
| points | int | 消耗积分 |
| created_at | datetime | 创建时间 |

## 核销明细表 (xhbs_points_user_bill_detail)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | int PK | 主键 |
| into_bill_id | int FK | 关联入账单ID |
| out_bill_id | int FK | 关联出账单ID |
| points | int | 核销积分 |
| created_at | datetime | 创建时间 |
