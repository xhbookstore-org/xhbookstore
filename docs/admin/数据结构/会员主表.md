# 会员主表 (member)

## 说明
会员核心信息表，记录每个会员的基本信息、卡号、会员类型、有效期、积分等。

## 字段定义
| 字段 | 类型 | 约束 | 是否必填 | 说明 | 来源 |
|------|------|------|----------|------|------|
| id | int PK AUTO_INCREMENT | — | 自动 | 会员记录ID | 系统生成 |
| card_no | varchar(50) | UNIQUE NOT NULL | 是 | 会员卡号（11位数字） | 后端自动生成 |
| name | varchar(50) | — | 是 | 姓名 | 手动录入/Excel导入 |
| phone | varchar(20) | INDEX | 是 | 手机号（登录/查重） | 手动录入/Excel导入 |
| card_type_id | int | FK→card_type | 是 | 卡类型ID | 下拉选择 |
| level_id | int | FK→member_level | 否 | 会员级别ID | 系统设置 |
| dept_id | bigint(20) | FK→sys_dept | 是 | 开通门店ID | 当前登录员工门店 |
| valid_date | date | INDEX | 是 | 有效日期（到期日） | 后端自动计算 |
| status | tinyint | DEFAULT 0 | 是 | 0-正常 1-注销 2-挂失 | 系统默认/操作 |
| borrow_count_valid | int | DEFAULT 0 | 是 | 有效期内借阅次数 | 系统默认0 |
| current_points | int | DEFAULT 0 | 是 | 当前可用积分 | 积分系统维护 |
| remark | text | — | 否 | 备注（含单位名） | 手动录入/Excel导入 |
| last_operator | varchar(50) | — | 是 | 最近操作人 | 系统自动记录 |
| source | varchar(20) | DEFAULT 'manual' | 是 | manual/excel/wechat/api | 系统自动标记 |
| sync_erp | tinyint(1) | DEFAULT 0 | 是 | 是否已同步ERP | 系统自动 |
| created_at | datetime | — | 是 | 创建时间 | 系统自动 |
| updated_at | datetime | — | 是 | 更新时间 | 系统自动 |

## 状态枚举
| 值 | 含义 |
|----|------|
| 0 | 正常 |
| 1 | 注销 |
| 2 | 挂失 |

## 数据来源枚举
| 值 | 含义 |
|----|------|
| manual | 手动新增 |
| excel | 批量导入 |
| wechat | 微信注册 |
| api | API接口创建 |

## 关联关系
- card_type_id → card_type.id
- level_id → member_level.id
- dept_id → sys_dept.dept_id
- id → member_ext.member_id（1:1）
- id → points_history.member_id（1:N）
- id → book_borrow_order.member_id（1:N）
