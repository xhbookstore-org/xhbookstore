# 会员级别表 (member_level)

## 说明
定义会员的等级体系，不同等级对应不同的折扣率。

## 字段定义
| 字段 | 类型 | 约束 | 是否必填 | 说明 | 来源 |
|------|------|------|----------|------|------|
| id | int PK | — | 自动 | 主键 | 系统生成 |
| level_name | varchar(50) | — | 是 | 级别名称 | 系统初始化 |
| discount | decimal(5,2) | — | 是 | 折扣率 | 系统设置 |
| status | tinyint | — | 是 | 状态（0-正常，1-停用） | 系统管理 |

## 关联关系
- id → member.level_id
