# 系统基础表

管理后台使用的系统基础表，来自RuoYi框架。

## 用户表 (sys_user)
| 字段 | 类型 | 说明 |
|------|------|------|
| user_id | bigint PK | 用户ID |
| dept_id | bigint | 部门ID |
| user_name | varchar(30) | 登录账号 |
| nick_name | varchar(30) | 昵称 |
| password | varchar(100) | 密码(加密) |
| phonenumber | varchar(11) | 手机号 |
| email | varchar(50) | 邮箱 |
| sex | char(1) | 性别 |
| avatar | varchar(100) | 头像URL |
| status | char(1) | 状态(0正常1停用) |
| login_ip | varchar(128) | 最后登录IP |
| login_date | datetime | 最后登录时间 |
| del_flag | char(1) | 删除标识 |

## 部门表 (sys_dept) — 同时作为门店数据
| 字段 | 类型 | 说明 |
|------|------|------|
| dept_id | bigint PK | 部门/门店ID |
| parent_id | bigint | 上级ID |
| dept_name | varchar(30) | 部门/门店名称 |
| leader | varchar(20) | 负责人 |
| phone | varchar(11) | 电话 |
| email | varchar(50) | 邮箱 |
| order_num | int | 排序 |
| status | char(1) | 状态 |

## 角色表 (sys_role)
| 字段 | 类型 | 说明 |
|------|------|------|
| role_id | bigint PK | 角色ID |
| role_name | varchar(30) | 角色名称 |
| role_key | varchar(100) | 权限字符 |
| role_sort | int | 排序 |
| data_scope | char(1) | 数据范围 |
| status | char(1) | 状态 |

## 菜单表 (sys_menu)
| 字段 | 类型 | 说明 |
|------|------|------|
| menu_id | bigint PK | 菜单ID |
| parent_id | bigint | 父菜单ID |
| menu_name | varchar(50) | 菜单名称 |
| menu_type | char(1) | M目录/C菜单/F按钮 |
| path | varchar(200) | 路由地址 |
| component | varchar(255) | 组件路径 |
| perms | varchar(100) | 权限标识 |
| icon | varchar(100) | 图标 |
| order_num | int | 排序 |
| visible | char(1) | 是否显示 |
| status | char(1) | 状态 |

## 关联表
- sys_user_role：用户-角色关联(user_id, role_id)
- sys_role_menu：角色-菜单关联(role_id, menu_id)
