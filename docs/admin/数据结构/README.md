## 建表约定

- 业务表必须保留 `created_at`、`updated_at`、`is_del`。
- 删除默认逻辑删除。
- 涉及订单号、会员编号、卡号的字段必须建立唯一约束或明确去重策略。
- 涉及金额、状态、库存、积分的表必须能追溯操作人和业务来源。
