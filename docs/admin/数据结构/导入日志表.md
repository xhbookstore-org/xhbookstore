# 导入相关表

## 会员导入日志主表 (member_import_log)
记录每次批量导入的批次信息。

| 字段 | 类型 | 约束 | 是否必填 | 说明 | 来源 |
|------|------|------|----------|------|------|
| id | int PK | — | 自动 | 主键 | 系统生成 |
| operator | varchar(50) | NOT NULL | 是 | 操作人 | 当前登录用户 |
| import_time | datetime | — | 是 | 导入时间 | 系统自动 |
| file_name | varchar(255) | — | 是 | Excel文件名 | 上传文件 |
| total_records | int | — | 是 | 总条数 | 系统统计 |
| success_records | int | — | 是 | 成功条数 | 系统统计 |
| fail_records | int | — | 是 | 失败条数 | 系统统计 |
| original_content | longtext | — | 否 | 原始JSON | 系统快照 |
| error_log | text | — | 否 | 错误日志 | 系统生成 |
| remark | varchar(255) | — | 否 | 备注 | 手动输入 |
| created_at | datetime | — | 是 | 创建时间 | 系统自动 |

## 会员导入明细表 (member_import_detail)
记录批量导入的每一行数据处理结果。

| 字段 | 类型 | 约束 | 是否必填 | 说明 | 来源 |
|------|------|------|----------|------|------|
| id | int PK | — | 自动 | 主键 | 系统生成 |
| log_id | int | FK→member_import_log CASCADE | 是 | 关联日志ID | 系统关联 |
| row_index | int | — | 是 | Excel行号 | Excel解析 |
| card_no | varchar(50) | — | 否 | 卡号快照 | Excel解析 |
| name | varchar(50) | — | 否 | 姓名快照 | Excel解析 |
| phone | varchar(20) | — | 否 | 手机号快照 | Excel解析 |
| card_type_name | varchar(50) | — | 否 | 卡类型快照 | Excel解析 |
| level_name | varchar(50) | — | 否 | 级别快照 | Excel解析 |
| valid_date | date | — | 否 | 有效期快照 | Excel解析 |
| update_time | datetime | — | 否 | 修改时间快照 | Excel解析 |
| import_status | tinyint | — | 是 | 0-失败 1-成功 | 系统标记 |
| member_id | int | FK→member | 否 | 关联会员ID | 系统关联 |
| error_msg | varchar(500) | — | 否 | 失败原因 | 系统生成 |
| created_at | datetime | — | 是 | 创建时间 | 系统自动 |

## 导入流程
1. 上传Excel → 2. 逐行解析校验 → 3. member/member_ext写入 → 4. import_log记录批次 → 5. import_detail记录每行 → 6. 展示结果
