# 管理后台 CHANGELOG

## v20260705

### 会员卡业务

- 新增会员卡售卖业务：支持员工扫码会员动态码后开卡。
- 新增会员卡队列：同一会员可拥有多张卡，同一时间只生效一张，当前卡过期后自动激活下一张待生效卡。
- 新增会员卡退卡：以付款时间为准 7 天内可退，不以生效时间为准；退卡只能在 admin 后台操作。
- 新增售卡订单、退卡订单、会员卡业务日志，形成收入、退款、卡状态的对账闭环。

### 会员导入导出

- 新增会员 Excel 导入日志和导入明细落库。
- 导入结果区分成功、未导入、失败。
- 会员卡号或会员编号已存在时归为未导入。
- 系统不存在 Excel 中的卡类型时归为失败，不写入会员、不生成会员卡和订单。
- 导出按查询条件导出全量数据，文件名规则为 `门店会员_yyyyMMdd_HHmmss.xlsx` 或 `全部会员_yyyyMMdd_HHmmss.xlsx`。

### 后台页面

- 会员列表增加会员编号查询条件、会员卡入口、退卡入口。
- 新增会员卡记录、会员卡订单、会员退卡订单页面。
- 顶部导航隐藏通知、源码地址、文档地址、布局大小、全屏按钮和布局设置，增加当前登录用户部门展示。
