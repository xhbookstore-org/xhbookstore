# 登录流程

## 流程图

```
用户访问管理后台
    │
    ├─ 已登录且有有效Session ──→ 直接进入首页
    │
    └─ 未登录 ──→ 显示登录页
                    │
                    ▼
            输入账号 + 密码 + 验证码
                    │
                    ▼
              点击"登录"按钮
                    │
                    ▼
            前端校验（非空、格式）
                    │
              ┌─ 校验失败 ──→ 提示错误信息
              │
              ▼ 校验通过
            调用 POST /login
                    │
              ┌─ 接口返回失败 ──→ 提示错误 + 刷新验证码
              │   ├─ 账号密码错误 → "用户名或密码错误"
              │   ├─ 验证码错误 → "验证码错误"
              │   └─ 账号被停用 → "账号已被停用"
              │
              ▼ 接口返回成功
            获取 Token，存储到 localStorage
                    │
                    ▼
            调用 GET /getInfo
            获取用户信息、角色、权限集合
                    │
                    ▼
            调用 GET /getRouters
            获取用户有权限的路由菜单
                    │
                    ▼
            前端动态添加路由
            渲染侧边栏菜单
                    │
                    ▼
            跳转到首页仪表盘
```

## 登录状态维护

- 采用 SpringSecurity + Session Cookie 机制
- 服务端维护 HttpSession
- 前端每次请求自动携带 Cookie（浏览器默认行为）
- Session 过期后，请求返回 302，前端拦截跳转登录页

## 退出登录

```
用户点击"退出登录"
    │
    ▼
调用 POST /logout
    │
    ▼
清除前端 localStorage / Vuex 中的用户数据
    │
    ▼
清除服务端 Session
    │
    ▼
跳转到登录页
```
