## 关键规则

1. 会员卡退款以付款时间为准，7 天内可退，不按生效时间。
2. 退卡只能在 admin 后台操作，小程序端不提供退卡入口。
3. 同一会员可购买多张卡，但同一时刻只能一张卡生效。
4. 卡片状态刷新不依赖定时任务，查询或操作时发现当前卡已过期，则触发下一张待生效卡激活。
5. Excel 导入时，会员卡号、会员编号已存在算未导入，不算成功，也不算失败。
6. Excel 导入时，卡类型在系统中不存在算失败，不入会员、不入会员卡、不生成订单。
