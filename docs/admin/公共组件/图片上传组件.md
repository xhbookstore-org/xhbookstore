# 图片上传组件 (ImageUpload)

## 说明

基于 Element UI `el-upload` 封装的图片上传组件，支持预览和删除。

## 组件路径

`@/components/ImageUpload`

## 使用方式

```vue
<image-upload
  v-model="avatar"
  :limit="1"
/>
```

## Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| value / v-model | string | '' | 图片URL |
| limit | number | 1 | 最大上传数量 |
| fileSize | number | 5 | 文件大小限制（MB） |
| disabled | boolean | false | 是否禁用 |

## 认证方式

上传时自动携带 Cookie（SpringSecurity Session认证）。

## 限制

- 仅允许图片类型：`image/png`, `image/jpeg`, `image/gif`
- 最大文件大小：默认5MB
- 最大数量：通过 `limit` prop 控制

## 使用场景

| 场景 | 页面 |
|------|------|
| 用户头像上传 | C04-个人中心 |
| 图书封面/内页上传 | 图书管理（待建） |
| 借书/还书拍照上传 | 借阅管理（待建） |
