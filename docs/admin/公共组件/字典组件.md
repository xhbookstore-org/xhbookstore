# 字典组件

## 说明

字典相关的公共组件，用于统一处理字典数据的加载和展示。

---

## 一、DictData 组件

### 说明

从后端字典接口加载指定字典类型的数据，用于下拉框、单选框等。

### 组件路径

`@/components/DictData`

### 使用方式

```vue
<!-- 下拉框形式 -->
<dict-data v-model="status" dictType="sys_normal_disable" />

<!-- 通过 slot 自定义展示 -->
<dict-data dictType="sys_user_sex">
  <template #default="{ list }">
    <el-radio-group v-model="sex">
      <el-radio v-for="item in list" :key="item.dictValue" :label="item.dictValue">
        {{ item.dictLabel }}
      </el-radio>
    </el-radio-group>
  </template>
</dict-data>
```

### Props

| 属性 | 类型 | 必填 | 说明 |
|------|------|------|------|
| dictType | string | 是 | 字典类型标识 |
| value / v-model | string | 否 | 绑定值 |

---

## 二、DictTag 组件

### 说明

将字典值渲染为标签（Tag），通常用于表格列中。

### 组件路径

`@/components/DictTag`

### 使用方式

```vue
<dict-tag :options="statusOptions" :value="row.status" />
```

### Props

| 属性 | 类型 | 必填 | 说明 |
|------|------|------|------|
| options | array | 是 | 字典数据列表 |
| value | string/number | 是 | 当前值 |

---

## 三、字典数据获取

### 方式一：通过组件自动获取

DictData 组件会根据 `dictType` 自动请求接口加载数据。

### 方式二：手动加载

```javascript
import { getDicts } from '@/api/system/dict/data'

// 在 created/mounted 中
getDicts('sys_normal_disable').then(res => {
  this.statusOptions = res.data
})
```

---

## 常用字典类型

| 字典类型 | 说明 | 使用页面 |
|----------|------|----------|
| sys_normal_disable | 系统开关（正常/停用） | 全部管理页面 |
| sys_user_sex | 用户性别 | S01-用户管理 |
| sys_notice_type | 通知类型 | S08-通知公告 |
| sys_oper_type | 操作类型 | T06-操作日志 |
| sys_job_status | 任务状态 | T02-定时任务 |

## 注意

会员卡类型（年卡/半年卡/续费年卡/续费半年卡）不使用字典组件，而是通过独立的 `card_type` 表和 `/member/cardTypes` 接口获取。
