# 分页组件 (Pagination)

## 说明

基于 Element UI `el-pagination` 封装的分页组件，用于所有列表页面。

## 组件路径

`@/components/Pagination`

## 使用方式

```vue
<pagination
  v-show="total > 0"
  :total="total"
  :page.sync="queryParams.pageNo"
  :limit.sync="queryParams.pageSize"
  @pagination="getList"
/>
```

## Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| total | number | 0 | 总条数 |
| page | number | 1 | 当前页码（支持 .sync） |
| limit | number | 20 | 每页条数（支持 .sync） |
| pageSizes | number[] | [10, 20, 50, 100] | 每页条数选项 |
| layout | string | "total, sizes, prev, pager, next, jumper" | 分页布局 |

## 事件

| 事件名 | 参数 | 说明 |
|--------|------|------|
| pagination | — | 页码或每页条数变化时触发，重新请求列表 |

## 示例

```vue
<template>
  <div>
    <el-table :data="list" v-loading="loading">...</el-table>
    <pagination
      v-show="total > 0"
      :total="total"
      :page.sync="queryParams.pageNo"
      :limit.sync="queryParams.pageSize"
      @pagination="getList"
    />
  </div>
</template>
```
