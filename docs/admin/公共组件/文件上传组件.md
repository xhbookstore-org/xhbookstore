# 文件上传组件 (FileUpload)

## 说明

基于 Element UI `el-upload` 封装的文件上传组件，支持单文件和多文件上传。

## 组件路径

`@/components/FileUpload`

## 使用方式

```vue
<file-upload
  v-model="fileUrl"
  :fileSize="10"
  :fileType="['doc', 'docx', 'xls', 'xlsx', 'pdf']"
/>
```

## Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| value / v-model | string | '' | 文件URL |
| fileSize | number | 5 | 文件大小限制（MB） |
| fileType | string[] | — | 允许的文件类型列表 |

## 上传接口

`POST /common/upload`

## 限制

- 最大文件大小：默认5MB
- 文件类型：根据 `fileType` prop 限制
- 仅允许上传单个文件（`:limit="1"`）

## 会员批量导入

会员管理模块的批量导入功能使用独立的 Excel 上传弹窗，需额外处理：

1. 上传 Excel 文件
2. 后端解析 + 逐行校验
3. 返回导入结果（成功/失败条数、失败行详情）
4. 弹窗展示导入结果
